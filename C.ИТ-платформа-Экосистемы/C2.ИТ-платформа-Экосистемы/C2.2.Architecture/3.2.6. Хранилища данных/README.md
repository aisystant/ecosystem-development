# Хранилища данных экосистемы

## Назначение

Эта папка содержит описания основных репозиториев экосистемы, которые служат **хранилищами данных и знаний**. Эти репозитории являются частью технической архитектуры платформы и обеспечивают работу ИИ-агента **[[ИИ-агент Индексатор знаний 3.2|Индексатор знаний]]**.

---

## Основные хранилища данных

### 1. Репозитории знаний и документации

#### ecosystem-development
- **URL:** https://github.com/aisystant/ecosystem-development
- **Назначение:** Основное хранилище знаний экосистемы (этот репозиторий)
- **Тип:** Общедоступное хранилище методологии, архитектуры и процессов
- **Описание:** Содержит структурированные знания об экосистеме по системной модели (Мир, Созидатель, Экосистема развития)

#### docs
- **URL:** https://github.com/aisystant/docs
- **Назначение:** Публичная документация экосистемы
- **Тип:** Общедоступная документация для пользователей
- **Описание:** Содержит пользовательскую документацию, руководства, FAQ

- **URL:** https://github.com/aisystant/personal-knowledge
- **Назначение:** Персональный репозиторий знаний участников
- **Тип:** Шаблон для персональных хранилищ

- **Детальное описание:** [[3.2.6.1. Репозиторий знаний/Репозиторий знаний - структура и задание|Репозиторий знаний — структура и задание]]

### 2. MCP-серверы

Model Context Protocol (MCP) серверы обеспечивают интеграцию хранилищ данных с ИИ-системами.

#### fsm-mcp
- **URL:** https://github.com/aisystant/fsm-mcp
- **Назначение:** Логика работы Ассистента ученика — FSM-инструкции и переходы между состояниями
- **Тип:** MCP-сервер логики поведения
- **Описание:** Реализует конечный автомат (FSM) для управления поведением Ассистента ученика в образовательном процессе

#### digital-twin-mcp
- **URL:** https://github.com/aisystant/digital-twin-mcp
- **Назначение:** Доступ к данным цифрового двойника созидателя
- **Тип:** MCP-сервер данных
- **Описание:** Управляет персональными данными пользователя через цифровой двойник

#### guides-mcp
- **Назначение:** Доступ к репозиторию руководств для ДЗ-чекера и других агентов
- **Тип:** MCP-сервер данных
- **Описание:** Предоставляет доступ к образовательным руководствам и материалам

## Связь с Индексатором знаний

**[[ИИ-агент Индексатор знаний 3.2|Индексатор знаний]]** — ИИ-агент, который выполняет полный цикл работы с этими хранилищами:

1. **Сбор** — извлекает первичный контент из различных источников
2. **Нормализация** — преобразует контент в чанки (структурированные единицы до 10 000 знаков)
3. **Индексация** — создаёт векторные представления (эмбеддинги) для семантического поиска
4. **Хранение** — сохраняет данные в соответствующих хранилищах

**Поток данных:**
```
Первичный контент → Индексатор знаний → Репозиторий знаний → База эмбеддингов → ИИ-системы
```
