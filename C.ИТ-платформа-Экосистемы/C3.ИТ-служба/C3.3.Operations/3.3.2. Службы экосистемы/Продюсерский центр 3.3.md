# Продюсерский центр

## 0. Назначение документа

Этот документ описывает Продюсерский центр — службу экосистемы, которая помогает созидателям развивать публичность и становиться Просветителями. Центр разрабатывает и применяет практики раскрутки на основе инженерного подхода.

---

## 1. Зачем нужен Продюсерский центр

### 1.1. Проблема

Для работы на высоких системных уровнях необходимо быть известным или входить в элиту. Иначе люди не считывают эпистемологический статус созидателя.

**Эволюционная логика:**
- Люди учатся у «альфа-источников» — тех, кого считают лидерами
- Доверие к публичным людям выше, чем к малоизвестным профессионалам
- Это нормальное поведение, основанное на эвристиках выживания

**Следствие:** Даже выдающийся профессионал с минимальной публичностью имеет ограниченное влияние.

### 1.2. Формула известности

```
Известность = Значимое дело × Публичная раскрутка
```

**Значимое дело** — необходимое условие:
- Сделать что-то важное для большого числа людей
- Без этого публичность пуста и краткосрочна

**Публичная раскрутка** — достаточное условие:
- Иногда срабатывает случай (как с Эйнштейном после солнечного затмения 1919)
- Но случай непредсказуем
- Нужен инженерный подход

### 1.3. Миссия Продюсерского центра

**Разработать и применять практики раскрутки на основе инженерии:**
- Систематический подход вместо надежды на удачу
- Постоянное тестирование гипотез
- Накопление работающих практик
- Масштабирование успешных паттернов

## 2. Функции Продюсерского центра

### 2.1. Основные функции

| Функция | Описание |
|---------|----------|
| **Диагностика** | Оценка текущего уровня публичности и потенциала созидателя |
| **Стратегия** | Разработка персональной стратегии раскрутки |
| **Продакшн** | Производство контента: статьи, видео, подкасты, курсы |
| **Дистрибуция** | Размещение контента в правильных каналах |
| **Нетворкинг** | Выстраивание связей с другими экспертами и медиа |
| **Аналитика** | Измерение результатов и корректировка стратегии |

### 2.2. Кого обслуживает центр

**Целевая аудитория:**
- Созидатели уровня Исследователь, готовые к публичности
- Профессионалы с запросом на расширение влияния
- Внутренние эксперты экосистемы

**Критерии отбора:**
- Наличие значимого дела (метод, теория, проект)
- Готовность инвестировать время в публичность
- Системный подход к развитию
- Этическая позиция (не инфобизнес ради инфобизнеса)

---

## 3. Практики раскрутки

### 3.1. Инженерный подход

Раскрутка — это инженерная дисциплина, а не магия или удача.

**Принципы:**
1. **Гипотеза → Эксперимент → Анализ → Итерация**
2. **Измеримость:** всё, что делаем, должно быть измеримо
3. **Портфель гипотез:** одновременно тестируем несколько подходов
4. **Терпение:** результаты публичности накапливаются нелинейно

### 3.2. Каталог практик

#### Уровень 1: Базовая публичность

| Практика | Описание | Метрики |
|----------|----------|---------|
| **Личный сайт/блог** | Площадка для публикации экспертных материалов | Посещаемость, время на сайте |
| **Профиль в соцсетях** | Присутствие в 2-3 ключевых сетях | Подписчики, вовлечённость |
| **Регулярные публикации** | 1-2 материала в неделю | Количество, охват |
| **Экспертные комментарии** | Ответы на вопросы в профильных сообществах | Репутация, цитирования |

#### Уровень 2: Расширение охвата

| Практика | Описание | Метрики |
|----------|----------|---------|
| **Выступления** | Конференции, митапы, подкасты | Количество, размер аудитории |
| **Гостевые публикации** | Статьи на внешних площадках | Охват, переходы |
| **Коллаборации** | Совместные проекты с другими экспертами | Кросс-аудитория |
| **Медиа-появления** | Интервью, комментарии для СМИ | Упоминания, охват |

#### Уровень 3: Масштабирование влияния

| Практика | Описание | Метрики |
|----------|----------|---------|
| **Книга/курс** | Структурированная передача знаний | Продажи, отзывы |
| **Сообщество** | Создание группы последователей | Размер, активность |
| **Менторство публичных фигур** | Передача статуса через учеников | Успехи учеников |
| **Формирование повестки** | Определение тем обсуждения в области | Цитируемость |

### 3.3. Антипаттерны

**Чего избегать:**
1. **Публичность без дела.** Нет экспертизы → нет долгосрочного статуса
2. **Накрутки и манипуляции.** Разрушают репутацию при обнаружении
3. **Токсичные методы.** Хейт, скандалы — краткосрочная известность, долгосрочный ущерб
4. **Нерегулярность.** Публичность требует постоянства
5. **Игнорирование аналитики.** Без измерений нет улучшений

## 4. Процессы Продюсерского центра

### 4.1. Онбординг созидателя

```
1. Заявка → 2. Диагностика → 3. Стратегия → 4. Контракт → 5. Запуск
```

**Этапы:**
1. **Заявка.** Созидатель заполняет анкету: достижения, цели, ресурсы
2. **Диагностика.** Оценка текущей публичности, анализ ниши
3. **Стратегия.** Разработка персонального плана на 6-12 месяцев
4. **Контракт.** Согласование взаимных обязательств
5. **Запуск.** Начало работы по плану

### 4.2. Регулярная работа

**Еженедельно:**
- Планирование контента на неделю
- Производство 1-2 единиц контента
- Публикация и продвижение
- Анализ метрик предыдущей недели

**Ежемесячно:**
- Ретроспектива: что сработало, что нет
- Корректировка стратегии
- Планирование крупных активностей (выступления, коллаборации)

**Ежеквартально:**
- Стратегический обзор
- Оценка прогресса к целям
- Пересмотр позиционирования при необходимости

### 4.3. Производство контента

**Типы контента:**
- Статьи и посты (короткие, длинные)
- Видео (образовательное, экспертное, личное)
- Подкасты и аудио
- Презентации и выступления
- Курсы и программы

**Процесс производства:**
1. Идея → 2. Сценарий/план → 3. Создание → 4. Редактура → 5. Публикация → 6. Продвижение

## 6. Метрики Продюсерского центра

### 6.1. Метрики созидателя

| Метрика | Описание | Целевые значения |
|---------|----------|------------------|
| **Охват** | Суммарная аудитория публикаций | Рост 20%+ в квартал |
| **Вовлечённость** | Лайки, комментарии, репосты | >5% от охвата |
| **Подписчики** | Общее число подписчиков | Рост 10%+ в месяц |
| **Входящие запросы** | Приглашения, предложения, вопросы | Рост 30%+ в квартал |
| **Цитируемость** | Упоминания другими экспертами | Наличие и рост |
| **Конверсии** | Переходы в продукты/услуги | По индивидуальному плану |

### 6.2. Метрики центра

| Метрика | Описание |
|---------|----------|
| **Число активных созидателей** | Сколько созидателей ведёт центр |
| **Успешные кейсы** | Созидатели, достигшие целей |
| **Новые практики** | Количество разработанных практик |
| **Удовлетворённость** | NPS созидателей |

## 8. Развитие Продюсерского центра

### 8.1. Краткосрочные задачи

- Сформировать каталог базовых практик раскрутки
- Запустить пилот с 3-5 созидателями
- Разработать шаблоны контента и процессов
- Настроить аналитику

### 8.2. Среднесрочные задачи

- Масштабировать на 20+ созидателей
- Создать внутреннюю медиа-команду
- Разработать программу обучения продюсеров
- Накопить базу успешных кейсов

### 8.3. Долгосрочное видение

Продюсерский центр становится системой, которая:
- Превращает Исследователей в Просветителей
- Формирует пул публичных экспертов экосистемы
- Влияет на повестку в образовании и развитии
- Является узнаваемым брендом в области продюсирования экспертов
