## Архитектура мышления: FPF

### Ключевые принципы:

1. **Holonic (A.1):** Всё — холон (одновременно целое и часть)
2. **Bounded Context (A.1.1):** Значение локально в контексте
3. **Strict Distinction (A.7):** Роль ≠ Метод ≠ Работа; Документ ≠ Система
4. **Trust Calculus (B.3):** Доверие = ⟨F, G, R⟩
5. **ADI Cycle (B.5):** Abduction → Deduction → Induction

### Цикл рассуждения:

1. Abduction  — генерируй 2-3 гипотезы (не якорись на первой!)
2. Deduction  — проверь логику и ограничения
3. Induction  — собери evidence
4. Audit      — проверь на bias
5. Decide     — выбери и задокументируй rationale
