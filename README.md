# ecosystem-development
Канонический репозиторий экосистемы: идеи изменения мира, целевые системы для людей, и метасистема их создания.
## Слои
- `content/` — **человеческий слой** (источник истины). Здесь всё, что мыслится, редактируется и утверждается человеком.    
- `artifacts/` — **производный слой** (черновики ИИ, отчёты, сводки, сгенерённые схемы). Всё, что можно пересобрать.    
- `agents-core/` — **исполняемый слой агентов** (манифесты и код их инструментов).    
- `content/contracts/` — **контракты/схемы** (правила обмена, форматы входов/выходов, JSON-схемы).    
- `content/templates/` — **шаблоны** (человеко-ориентированные: паспорта агентов, ADR, отчёты).
## Правила
- `main` — защищена: изменения только через PR.
- Агенты создают ветки `ai/<agent>/<YYYY-MM-DD>` → PR в `main`.
- Только человек переносит утверждённое из `artifacts/` в `content/`.

## Поток
Obsidian → `content/` → (Агенты) → `artifacts/` → PR → ревью → `content/`.

## Родственные репозитории
- `ecosystem-development` (этот, канон)
- `agents-core` (код агентов / manifests)
- `ecosystem-platform` (исполняемые сервисы)
