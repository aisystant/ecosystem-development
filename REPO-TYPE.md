# Тип репозитория

**Тип**: `Downstream/governance`

**Source-of-truth**: no

## Роль

Управленческая downstream-сборка для экосистемы развития интеллекта. Содержит проектную/управленческую информацию, планирование, трассировку решений, навигацию. Оформлен по стандарту s2r.

## Upstream dependencies

- [TserenTserenov/spf-ecosystem-pack](https://github.com/TserenTserenov/spf-ecosystem-pack) — source-of-truth области экосистемы
- [aisystant/spf-personal-pack](https://github.com/aisystant/spf-personal-pack) — source-of-truth области созидателя
- [TserenTserenov/spf-digital-platform-pack](https://github.com/TserenTserenov/spf-digital-platform-pack) — source-of-truth области ИТ-платформы
- [TserenTserenov/s2r](https://github.com/TserenTserenov/s2r) — формат оформления
- [TserenTserenov/SPF](https://github.com/TserenTserenov/SPF) — фреймворк вторых принципов

## Downstream outputs

- Координация всех downstream-проектов экосистемы
- Реестр репозиториев (REPOSITORY-REGISTRY.md)

## Non-goals

- НЕ является source-of-truth предметных областей (это pack'и)
- НЕ содержит кода, сервисов
- НЕ определяет «что истинно» — только организует работу

## Особая роль

Содержит **реестр всех репозиториев экосистемы** в `/0.OPS/REPOSITORY-REGISTRY.md`
