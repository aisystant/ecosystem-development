# О разделе 0.0

## 0. Назначение документа

Этот документ описывает назначение раздела «0. Управление» — здесь фиксируются логика хранилища, процессы работы, шаблоны, автоматические отчёты и входящие материалы.

---

## 1. О чём этот раздел

Раздел «Управление» — это «панель управления» всего хранилища. Здесь:
- Определяется логика и структура хранилища (0.1)
- Описываются процессы работы (0.2)
- Хранятся шаблоны документов (0.3)
- Генерируются автоматические отчёты ИИ (0.4)
- Принимаются входящие материалы (0.9)

---

## 2. Структура раздела

| Подраздел | Назначение |
|-----------|------------|
| 0.1. Логика хранилища и знаний | Статическая информация: структура, принципы, глоссарий |
| 0.2. Процессы работы с хранилищем | Динамическая информация: как работать, ADR, план работ |
| 0.3. Шаблоны | Шаблоны для создания новых документов |
| 0.4. Автоматические отчёты ИИ | Отчёты, генерируемые ИИ-ассистентами |
| 0.9. Входящие | Новые предложения и черновики |
