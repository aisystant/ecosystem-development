## Как работать с классификацией

1. **Автоматическая классификация**: запустите `python ops/classify_documents.py`
2. **Ручная правка**:
   - Измените значения в таблице напрямую
   - Уберите тег `<mark>` для подтверждения правки
   - ⚠️ **Важно**: Можно выбирать ТОЛЬКО значения из списков выше
3. **Проверка валидности**: запустите `python ops/validate_classifications.py`
   - Скрипт проверит, что все значения соответствуют документу 0.7
   - GitHub Actions автоматически проверит при каждом коммите
4. **Новые документы**: автоматически появятся при следующем запуске скрипта
5. **Изменения в 0.7**: при изменении списка тегов скрипт предложит обновления
