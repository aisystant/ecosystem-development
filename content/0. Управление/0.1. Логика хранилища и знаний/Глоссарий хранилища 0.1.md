# Глоссарий хранилища 0.1

## 0. Назначение документа

Этот документ содержит определения ключевых терминов, используемых в хранилище экосистемы.

---

## 1. Термины хранилища

| Термин | Определение |
|--------|-------------|
| **Ячейка** | Место в структуре хранилища, определяемое уровнем системы и ролью мышления |
| **Семейство документов** | Группа документов F0-F9, объединённых по системному уровню и роли |
| **Frontmatter** | YAML-блок в начале документа с метаданными |
| **ADR** | Architecture Decision Record — запись о принятом архитектурном решении |

---

## 2. Термины системной модели

| Термин | Определение |
|--------|-------------|
| **Надсистема (Мир)** | Контекст, в который встраивается созидатель |
| **Целевая система (Созидатель)** | Человек, которого развивает экосистема |
| **Система создания (Экосистема)** | Платформа и команда, создающие созидателей |
