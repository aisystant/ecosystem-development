# О разделе 0.4

## 0. Назначение документа

Этот документ описывает назначение и содержание подраздела «0.4. Автоматические отчёты ИИ».

---

## 1. Назначение подраздела

Подраздел 0.4 содержит **автоматически генерируемые отчёты** от ИИ-ассистентов:

- Проверка полноты хранилища
- Анализ противоречий и несогласованностей
- Карты связей между документами
- Рекомендации по развитию

---

## 3. Принцип работы

1. ИИ-агенты (ConsistencyChecker, Librarian) анализируют хранилище
2. Результаты записываются в документы подраздела 0.4
3. Команда просматривает отчёты и принимает решения
4. Решения фиксируются как ADR в подразделе 0.9

---

## 4. Связь с другими подразделами

- **0.1** — логика хранилища (критерии для проверок)
- **0.3.1** — структура отчётов (шаблоны для генерации)
- **0.9** — входящие (сюда попадают вопросы из отчётов)
- **3.2.4** — ИИ-ассистенты (агенты, создающие отчёты)
