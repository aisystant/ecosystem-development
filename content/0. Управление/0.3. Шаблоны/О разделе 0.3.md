# О разделе 0.3

## 0. Назначение документа

Этот документ описывает назначение и содержание подраздела «0.3. Шаблоны».

---

## 1. Назначение подраздела

Подраздел 0.3 содержит **шаблоны документов** для обеспечения единообразия:

- Стандартные структуры для разных типов документов
- Примеры правильного оформления
- Подподразделы с узкоспециализированными шаблонами
