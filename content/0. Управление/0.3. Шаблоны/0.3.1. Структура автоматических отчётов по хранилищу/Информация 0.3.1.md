# Информация 0.3.1 — Структура автоматических отчётов по хранилищу

## Назначение раздела

Этот раздел содержит описания структур всех автоматических отчётов, которые генерируются ИИ-ассистентами для анализа состояния хранилища.

## Отличие от раздела 0.4

| Раздел | Содержание |
|--------|------------|
| **0.3.1** (этот раздел) | Шаблоны и описания структур отчётов |
| **0.4** | Сами отчёты с актуальными данными |

## Принцип работы

1. Структура каждого отчёта описывается в этом разделе (0.3.1)
2. ИИ-ассистент генерирует отчёт по описанной структуре
3. Готовый отчёт сохраняется в раздел 0.4
