# О папке Входящие

Эта папка предназначена для временных документов:
- Идеи для обсуждения
- Предложения на рассмотрение
- Вопросы, требующие решения

## Правила работы с папкой

1. Документы должны быть обработаны в течение 2 недель
2. После обсуждения документ либо переносится в нужную ячейку, либо удаляется
3. ИИ формирует отчёт о «зависших» документах в 0.4
