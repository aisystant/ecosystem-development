# Бэклог Релиза 1 — Демо без идентификации

Подробный план работ по реализации первого релиза Проводника (Ассистента ученика).

---

## Цель релиза

Работающий прототип в ChatGPT, который показывает недельные данные демо-пользователя и даёт рекомендации на основе методологии.

**Ключевое ограничение:** без идентификации. MCP-сервер работает с фиксированным демо-двойником.

## Зависимости между задачами

```
Документация (1.3-1.10) ─────────────────────────────────┐
                                                          │
UI (2.1-2.8) ──────────────────┐                         │
                                │                         │
MCP-сервер (3.1-3.6) ──────────┼───► Подключение (5.1-5.7)
                                │
Цифровой двойник (4.1-4.8) ────┘
```

1. Контракты MCP (1.6-1.9) должны быть готовы до реализации MCP-сервера (3.1-3.4) и UI (2.6)
2. Демо-данные в ЦД (4.2-4.4) должны быть готовы до тестирования MCP-сервера
3. UI и MCP должны быть готовы до подключения в ChatGPT
