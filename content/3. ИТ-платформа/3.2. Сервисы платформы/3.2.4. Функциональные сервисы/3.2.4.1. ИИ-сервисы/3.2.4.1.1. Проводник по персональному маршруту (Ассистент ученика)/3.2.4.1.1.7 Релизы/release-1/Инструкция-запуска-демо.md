# Инструкция запуска демо Release 1

## Предварительные требования

1. Доступ к ChatGPT Plus или Team
2. Приложение "Ассистент Ученика" подключено (настроено в OpenAI Developer Platform)
3. MCP-сервер развёрнут и доступен

---

## Шаги запуска

### 1. Открыть ChatGPT

Перейти на https://chat.openai.com

### 2. Открыть приложение

1. Нажать на "+" или "Explore GPTs"
2. Найти "Ассистент Ученика" (или использовать прямую ссылку)
3. Открыть приложение

### 3. Проверить подключение

После открытия должен загрузиться UI с экраном "Моя неделя".

Если UI не загрузился:
- Проверить консоль DevTools (F12 → Console)
- Проверить доступность MCP-сервера

### 4. Переключить демо-сценарий

Для переключения между сценариями (пустая / слабая / хорошая неделя):

**Если реализован вариант А (URL-параметр):**
- Пустая неделя: добавить `?demo=empty`
- Слабая неделя: добавить `?demo=weak`
- Хорошая неделя: добавить `?demo=good`

**Если реализован вариант Б (кнопки):**
- Найти селектор "Демо-профиль" в UI
- Выбрать нужный сценарий

### 5. Запросить рекомендацию

Написать в чат:
```
Покажи мою неделю и дай рекомендацию
```

или

```
Что мне делать сегодня?
```

### 6. Проверить результат

Сравнить ответ ассистента с ожидаемым (см. [[Демо-сценарии-данных-недели]]).

---

## Устранение неполадок

### UI не загружается

1. Проверить, что приложение опубликовано и доступно
2. Очистить кэш браузера
3. Попробовать в инкогнито-режиме

### MCP не отвечает

1. Проверить доступность эндпоинта:
   ```
   curl https://digital-twin-mcp.aisystant.workers.dev/mcp
   ```
2. Проверить логи в Cloudflare Workers Dashboard

### Данные не отображаются

1. Проверить, что демо-данные загружены в ЦД
2. Проверить ответ MCP в Network tab DevTools

### Рекомендация странная или пустая

1. Проверить, что методология ассистента настроена
2. Проверить формат ответа MCP-инструментов
