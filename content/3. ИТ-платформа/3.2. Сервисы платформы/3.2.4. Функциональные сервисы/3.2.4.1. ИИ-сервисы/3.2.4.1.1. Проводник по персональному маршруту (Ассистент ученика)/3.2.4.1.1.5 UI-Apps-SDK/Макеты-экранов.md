# Макеты экранов

## Принципы дизайна

1. **Минимализм** — только необходимые элементы
2. **Читаемость** — крупный шрифт, контрастные цвета
3. **Один экран** — без прокрутки для основной информации
4. **Консистентность** — единый стиль с ChatGPT

## Цветовая схема

| Элемент | Цвет | Назначение |
|---------|------|------------|
| Фон | #FFFFFF | Основной фон |
| Текст | #1A1A1A | Основной текст |
| Акцент | #10A37F | Кнопки, ссылки (цвет ChatGPT) |
| Серый | #6E6E80 | Вторичный текст |
| Успех | #10A37F | Положительные индикаторы |
| Ошибка | #EF4444 | Ошибки |
