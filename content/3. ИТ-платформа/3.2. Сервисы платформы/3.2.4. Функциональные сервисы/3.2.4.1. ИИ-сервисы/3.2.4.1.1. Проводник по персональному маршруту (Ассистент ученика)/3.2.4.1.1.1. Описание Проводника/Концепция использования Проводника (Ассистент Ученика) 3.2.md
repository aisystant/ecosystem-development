# Концепция использования Проводника (Ассистент Ученика) 3.2

## Пользовательский сценарий

### Типичная сессия

1. Пользователь открывает ChatGPT
2. Выбирает приложение "Проводник" (Aisystant Guide)
3. Нажимает "Показать мою неделю"
4. Ассистент запрашивает данные из Цифрового двойника через MCP
5. Отображает сводку: дни активности, время, рабочие продукты
6. Даёт рекомендации на основе данных

### Частота использования

- **Рекомендуемая:** 1-2 раза в неделю
- **Оптимальное время:** конец недели (пт-вс) для рефлексии
- **Минимальная:** раз в месяц для отслеживания трендов

## Точки входа

| Точка входа | Описание |
|-------------|----------|
| ChatGPT Apps | Основной способ — через каталог приложений |
| Прямая ссылка | URL для быстрого доступа |
| QR-код | Для мобильных пользователей |

## Ожидаемый результат

После сессии пользователь:
- Знает, сколько дней был активен
- Видит, сколько времени инвестировал в развитие
- Понимает, какие рабочие продукты создал
- Получил 1-2 конкретные рекомендации

## Ограничения

- Работает только с данными текущей недели (Release 1)
- Не показывает историю за прошлые периоды
- Не сравнивает с другими пользователями
- Не даёт оценок "хорошо/плохо" — только факты и рекомендации
