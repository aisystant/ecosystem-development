# Описание ИИ-ассистента Проводник (Ассистент Ученика) 3.2

## Что это

**Проводник** — ИИ-ассистент, который сопровождает ученика по персональной траектории развития. Работает внутри ChatGPT через OpenAI Apps SDK и получает данные из Цифрового двойника пользователя через MCP.

## Зачем нужен

1. **Отслеживание прогресса** — показывает ученику его активность за неделю
2. **Рефлексия** — помогает осознать, что было сделано и что можно улучшить
3. **Мотивация** — даёт конкретные рекомендации по следующим шагам
4. **Персонализация** — адаптирует советы под данные конкретного пользователя

## Для кого

- Ученики курсов Aisystant
- Пользователи, отслеживающие своё развитие через платформу
- Любой, кто хочет видеть свою недельную активность в удобном формате

## Ключевые характеристики

| Характеристика | Значение |
|----------------|----------|
| Платформа | ChatGPT (OpenAI Apps SDK) |
| Источник данных | Цифровой двойник через MCP |
| Период анализа | Текущая неделя (пн-вс) |
| Формат ответа | Один экран, структурированный |

## Что НЕ делает Проводник

- Не ведёт учёт (это делает Цифровой двойник)
- Не обучает (это делают курсы)
- Не хранит данные (только читает через MCP)
- Не принимает решения за пользователя
