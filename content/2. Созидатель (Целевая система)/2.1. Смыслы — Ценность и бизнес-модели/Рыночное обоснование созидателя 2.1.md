# Рыночное обоснование созидателя

## 0. Назначение документа

Этот документ обосновывает **рыночный потенциал созидателя** — почему будет спрос, какова приемлемая цена, и как созидатель позиционируется относительно альтернатив. Документ отвечает на вопросы: «Почему люди будут платить за то, чтобы стать созидателями?» и «Сколько это может стоить?»

## 3. Ценообразование

### 3.1. Референсные цены на рынке

| Продукт/Сервис | Цена | Что получает клиент |
|----------------|------|---------------------|
| Skillbox/Нетология (профессия) | $800-1500 | Курс 6-12 мес, диплом |
| MBA (онлайн) | $15000-50000 | 2 года, степень |
| Executive coaching | $300-1000/час | Индивидуальная работа |
| Подписка на обучение (Coursera+) | $50-100/мес | Доступ к контенту |
| Мастермайнд-группы | $200-500/мес | Сообщество + встречи |

### 3.2. Ценовая модель созидателя

**Принцип:** Цена отражает не «доступ к контенту», а **результат трансформации**.

| Уровень программы | Цена | Что получает | ROI для клиента |
|-------------------|------|--------------|-----------------|
| **Старт** (Ученик) | $300-500/квартал | Диагностика + маршрут + базовые руководства | Понятный путь развития |
| **Развитие** (Интеллектуал) | $800-1500/квартал | Стажировки + менторинг + сообщество | +20-50% к доходу за год |
| **Мастерство** (Профессионал) | $2000-4000/квартал | Резидентуры + проекты + экспертиза | +50-100% к доходу |
| **Исследование** | По запросу | Индивидуальная программа | Уникальная экспертиза |

### 3.3. Обоснование цены

**Почему $300-500 за квартал — приемлемая цена для старта?**

| Сравнение | Цена | Результат |
|-----------|------|-----------|
| 3 онлайн-курса | $300-500 | Контент без системы |
| 1 консультация коуча | $200-400 | Разовый разговор |
| Книги (10 шт) | $150-300 | Теория без практики |
| **Созидатель (старт)** | **$300-500** | **Система + маршрут + ритм** |

**Расчёт ROI для клиента:**

```
Инвестиция:     $500/квартал = $2000/год
Рост дохода:    +20% при зарплате $3000/мес = +$7200/год
ROI:            ($7200 - $2000) / $2000 = 260%
```

### 4.1. Карта конкурентов

```
                          ▲

         (дорого)         │        (уникально)
                          │
   ──────────────────────●┼───────────────────────►
   ТЕОРИЯ                 │                   ПРАКТИКА
                          │
         Курсы        ●───┼────●  Стажировки
         (массовые)       │      (узкие)
                          │
                          ▼
                      МАССОВОСТЬ
```

### 4.2. Сравнение с альтернативами

| Критерий | Онлайн-курсы | MBA | Коучинг | Созидатель |
|----------|--------------|-----|---------|------------|
| Персонализация | Низкая | Низкая | Высокая | Высокая |
| Практичность | Низкая | Средняя | Средняя | Высокая |
| Системность | Низкая | Высокая | Низкая | Высокая |
| Цена | Низкая | Высокая | Высокая | Средняя |
| Время | Гибко | 2 года | Гибко | Гибко |
| Сообщество | Слабое | Сильное | Нет | Сильное |
| Артефакты | Диплом | Степень | Нет | Портфолио |
| ИИ-интеграция | Нет | Нет | Нет | Да |

### 4.3. Уникальное позиционирование

**Созидатель занимает пустующую нишу:**

> «Персонализированное развитие системного мышления с практикой в реальных проектах, поддержкой ИИ-инструментов и сообщества — по цене, доступной среднему профессионалу»

**Ключевые отличия:**
1. **Артефакты вместо дипломов** — доказательства навыков
2. **Ритм вместо курса** — непрерывное развитие
3. **ИИ как инструмент** — не замена мышлению
4. **Сообщество практиков** — не студентов

## 5. Драйверы спроса

### 5.1. Макротренды

| Тренд | Влияние на спрос | Горизонт |
|-------|------------------|----------|
| Рост ИИ | Критическое — нужно учиться думать с ИИ | Сейчас |
| Удалённая работа | Высокое — нужны самостоятельные специалисты | 3-5 лет |
| Обесценивание дипломов | Среднее — рост ценности портфолио | 5-10 лет |
| Complexity crisis | Высокое — нужны системные мыслители | Долгосрочно |
| Lifelong learning норма | Высокое — готовность инвестировать в себя | 5-10 лет |

|---------|---------|-----------|

## 6. Барьеры и возражения

### 6.1. Основные возражения

| Возражение | Ответ | Доказательство |
|------------|-------|----------------|
| «Дорого» | ROI 260%+ за год при минимальном росте дохода | Расчёт в разделе 3.3 |
| «Нет времени» | 2-4 часа/неделю, встроенные в рабочий ритм | Формат слотов |
| «Ещё один курс» | Не курс — система развития с артефактами | Отличие от курсов |
| «Справлюсь сам» | Без структуры и сообщества — dropout 90%+ | Статистика самообучения |
| «Не понимаю результат» | Конкретные артефакты + рост дохода | Кейсы выпускников |

### 6.2. Барьеры входа

| Барьер | Решение |
|--------|---------|
| Неизвестный бренд | Контент-маркетинг, публичные кейсы |
| Сложно объяснить продукт | Бесплатная диагностика, trial |
| Страх не справиться | Гарантия возврата, поддержка |
| Нужно согласие семьи | Материалы для семьи, совместные форматы |

## 7. Юнит-экономика

### 7.1. Модель для одного созидателя

| Метрика | Значение | Комментарий |
|---------|----------|-------------|
| **CAC** (Customer Acquisition Cost) | $50-150 | Контент + реклама |
| **ARPU** (Average Revenue Per User) | $1000-2000/год | Подписка + программы |
| **LTV** (Lifetime Value) | $3000-8000 | При retention 2-4 года |
| **LTV/CAC** | 20-50x | Очень здоровая экономика |
| **Payback** | 1-2 месяца | Быстрая окупаемость |

### 7.2. Retention модель

```
Месяц    1    3    6    12   24   36
─────────────────────────────────────
Ученик   100% 70%  50%  35%  20%  10%
Интел-л  100% 85%  75%  60%  45%  30%
Проф-л   100% 90%  85%  80%  70%  60%
```

**Ключ к retention:** Переход на следующий уровень удерживает лучше, чем «больше контента».
