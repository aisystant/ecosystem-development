# Модель характеристик созидателя

## 0. Назначение документа

Этот документ описывает модель характеристик созидателя — набор измеримых свойств, которые определяют состояние и потенциал человека.

---

## 1. Концепция модели

Модель характеристик устанавливает связь:

```
Характеристика → Показатель → Значение → Состояние
```

**Характеристика** — это измеримое свойство системы (созидателя).

**Показатель** — конкретная метрика для измерения характеристики.

**Значение** — текущее значение показателя.

**Состояние** — совокупность значений всех показателей в данный момент.

---

## 2. Группы характеристик

### 2.1. Киберхарактеристики (основные)

**6 ключевых характеристик** для инженерного проектирования развития:

| Характеристика | Описание |
|----------------|----------|
| **Агентность** | Способность быть источником действия |
| **Интеллектуальная развитость** | Инженерия познания |
| **Стрессоустойчивость** | Устойчивость под давлением |
| **Ресурсность** | Доступ к времени, финансам, социальному капиталу |
| **Техноинтеграция** | Интеграция с экзотелом (включая экзокортекс) |
| **Калибр личности** | Масштаб замыслов и зона влияния |

## 3. Использование модели

### 3.1. Для диагностики

Оценка текущего состояния созидателя по всем характеристикам.

### 3.3. Для цифрового двойника

Структура данных для хранения и отслеживания характеристик.
