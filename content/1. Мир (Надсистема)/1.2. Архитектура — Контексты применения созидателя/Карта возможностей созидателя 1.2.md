# Карта возможностей созидателя

## 0. Назначение документа

Этот документ содержит **Opportunity-Solution Tree** — структурированную карту всех возможностей применения жизненного мастерства созидателя на разных системных уровнях.

**Методология:** Product Discovery (Teresa Torres)
**Формат:** Opportunity-Solution Tree

---

## 7. Матрица возможностей

## 8. Приоритизация возможностей

### 8.1. Критерии приоритизации

| Критерий | Описание |
|----------|----------|
| **Impact** | Какое влияние на жизнь созидателя и других? |
| **Fit** | Насколько соответствует способностям и контексту? |
| **Timing** | Подходящий ли момент? |
| **Resources** | Есть ли ресурсы (время, деньги, энергия)? |
| **Scalability** | Можно ли масштабировать? |

### 8.2. Рекомендуемая последовательность

```
1. ЛИЧНОСТЬ (фундамент)
   └── Self-management → Личные проекты

2. СЕМЬЯ (ближний круг)
   └── Роли → Обучение близких → Семейные системы

3. РАБОТА (экономическая основа)
   └── Сотрудник → Создатель систем → Предприниматель

4. ОБЩЕСТВО (расширение влияния)
   └── Сообщества → Гражданская позиция → Институты

5. ЦИВИЛИЗАЦИЯ (максимальный масштаб)
   └── Глобальные вызовы → Наследие
```
