# Созидатель как сотрудник

## 0. Назначение документа

Этот документ описывает, как созидатель **встраивается в организацию в роли сотрудника** — понимает корпоративную среду, играет профессиональную роль, создаёт ценность и развивается в рамках найма.

**Системный уровень:** Рабочие системы
**Позиция:** Созидатель как элемент организационной надсистемы

**Методология:** User Story Mapping, Service Blueprinting, Behavioral Prototyping
**Формат:** Сценарии применения и карьерные траектории

---

## 1. Организация как надсистема

### 1.1. Системный взгляд на организацию

Созидатель видит организацию как систему:
- **Элементы:** люди, команды, подразделения
- **Связи:** иерархия, коммуникации, процессы
- **Функции:** создание продукта/услуги, прибыль
- **Культура:** ценности, нормы, традиции
- **Среда:** рынок, конкуренты, регуляторы

## 2. Профессиональная роль созидателя

### 2.1. Компоненты роли

| Компонент | Описание |
|-----------|----------|
| **Позиция** | Формальное место в структуре |
| **Функции** | Что должен делать |
| **Полномочия** | Что может решать |
| **Ответственность** | За что отвечает |
| **Ожидания** | Что от него ждут |
| **Ресурсы** | Что дают для работы |

### 2.2. Осознанное принятие роли

Созидатель:
- **Изучает** — понимает роль до принятия
- **Договаривается** — уточняет ожидания
- **Принимает** — осознанно берёт обязательства
- **Исполняет** — качественно выполняет функции
- **Развивает** — расширяет роль по мере роста
- **Пересматривает** — регулярно сверяется с ожиданиями

## 3. Ключевые практики сотрудника-созидателя

### 3.1. Создание ценности

| Практика | Описание |
|----------|----------|
| **Результат** | Фокус на outcomes, а не activities |
| **Качество** | Высокий стандарт работы |
| **Эффективность** | Достигать больше с меньшими затратами |
| **Инициатива** | Предлагать улучшения |
| **Владение** | Брать ответственность за результат |

### 3.2. Работа в команде

| Практика | Описание |
|----------|----------|
| **Коллаборация** | Эффективно работать вместе |
| **Коммуникация** | Ясно доносить информацию |
| **Конфликты** | Решать разногласия конструктивно |
| **Поддержка** | Помогать коллегам |
| **Обратная связь** | Давать и принимать feedback |

### 3.3. Управление вверх

| Практика | Описание |
|----------|----------|
| **Прозрачность** | Держать руководителя в курсе |
| **Проактивность** | Не ждать указаний |
| **Alignment** | Понимать приоритеты руководства |
| **Влияние** | Продвигать свои идеи |
| **Доверие** | Выстраивать надёжные отношения |

### 3.4. Профессиональное развитие

| Практика | Описание |
|----------|----------|
| **Обучение** | Непрерывно осваивать новое |
| **Обратная связь** | Активно искать feedback |
| **Менторство** | Находить менторов и быть ментором |
| **Networking** | Строить профессиональные связи |
| **Portfolio** | Документировать достижения |

## 4. Навигация в организации

### 4.1. Понимание культуры

Созидатель изучает:
- **Явные правила** — политики, процессы, регламенты
- **Неявные нормы** — как «на самом деле» принято
- **Ценности** — что действительно важно
- **Власть** — кто принимает решения
- **Истории** — успехи и неудачи прошлого

### 4.2. Политическая грамотность

Созидатель умеет:
- Читать политическую карту организации
- Понимать интересы stakeholders
- Строить коалиции для продвижения идей
- Избегать деструктивных игр
- Защищать свои интересы этично

### 4.3. Управление границами

| Граница | Как управлять |
|---------|---------------|
| **Рабочее время** | Чёткие рамки, overtime осознанно |
| **Объём работы** | Говорить «нет» нереалистичному |
| **Токсичность** | Не участвовать в интригах |
| **Ценности** | Не компрометировать принципы |
| **Развитие** | Время на обучение |

## 5. Карьерные траектории

### 5.1. Типы карьерных треков

```
                    ┌─────────────────┐
                    │   МЕНЕДЖМЕНТ    │
                    │   (люди)        │
                    └────────▲────────┘
                             │
        ┌────────────────────┼────────────────────┐
        │                    │                    │
┌───────▼───────┐    ┌───────▼───────┐    ┌───────▼───────┐
│  ЭКСПЕРТИЗА   │    │   СТАРТ       │    │  ПРЕДПРИНИ-   │
│  (глубина)    │    │   КАРЬЕРЫ     │    │  МАТЕЛЬСТВО   │
└───────────────┘    └───────────────┘    └───────────────┘
```

### 5.2. Осознанный выбор траектории

Созидатель определяет:
- **Что драйвит** — деньги, влияние, мастерство, смысл?
- **Сильные стороны** — где максимальная отдача?
- **Ограничения** — что не готов делать?
- **Рынок** — какие возможности есть?
- **Тренды** — куда движется индустрия?

## 6. Типичные сценарии

### 6.1. Онбординг в новую компанию

**Контекст:** Первые 90 дней на новой работе.

**Что делает созидатель:**
1. **Слушает** — понимает контекст до действий
2. **Изучает** — культуру, процессы, людей
3. **Выстраивает связи** — знакомится с ключевыми людьми
4. **Показывает результат** — ранние победы
5. **Договаривается** — уточняет ожидания
6. **Адаптирует стиль** — под культуру организации

### 6.2. Работа с трудным руководителем

**Контекст:** Руководитель с проблемным стилем управления.

**Что делает созидатель:**
1. Понимает стиль и мотивацию руководителя
2. Адаптирует коммуникацию
3. Управляет ожиданиями проактивно
4. Документирует договорённости
5. Ищет поддержку (HR, ментор)
6. Оценивает: терпеть, менять ситуацию или уходить

### 6.3. Продвижение идеи

**Контекст:** Созидатель видит возможность улучшения.

**Что делает:**
1. Формулирует идею с фокусом на ценность для организации
2. Находит союзников
3. Готовит pitch с данными
4. Выбирает момент и формат
5. Предлагает пилот с минимальными рисками
6. Демонстрирует результаты

### 6.4. Переговоры о повышении

**Контекст:** Созидатель готов к новому уровню.

**Что делает:**
1. Собирает доказательства ценности
2. Изучает рынок и практики компании
3. Выбирает момент (после успехов, в бюджетный цикл)
4. Готовит аргументацию
5. Ведёт переговоры уверенно
6. Имеет Plan B

## 7. Антипаттерны

### 7.1. Чего избегать

| Антипаттерн | Проблема | Альтернатива |
|-------------|----------|--------------|
| **Пассивность** | Ждать указаний | Проактивность и инициатива |
| **Overwork** | Выгорание ради карьеры | Устойчивый темп |
| **Изоляция** | Работать только в своей роли | Networking и коллаборация |
| **Политиканство** | Интриги вместо результатов | Этичное влияние |
| **Стагнация** | Не развиваться | Непрерывное обучение |
| **Victim mentality** | Винить систему | Брать ответственность |

## 8. Результаты

### 8.1. Для созидателя

- Профессиональный рост и развитие
- Достойный доход
- Репутация и связи
- Реализация потенциала
- Подготовка к следующим этапам (предпринимательство, лидерство)

### 8.2. Для организации

- Высокая производительность
- Инновации и улучшения
- Развитие команды
- Культура excellence

### 8.3. Метрики

| Метрика | Описание |
|---------|----------|
| **Performance** | Результаты оценки работы |
| **Impact** | Влияние на бизнес-метрики |
| **Growth** | Рост в роли и ответственности |
| **Reputation** | Отзывы коллег и руководства |
| **Compensation** | Динамика вознаграждения |
