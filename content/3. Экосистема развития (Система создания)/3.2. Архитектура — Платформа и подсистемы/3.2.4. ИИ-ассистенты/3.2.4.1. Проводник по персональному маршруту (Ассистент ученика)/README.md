## Архитектура

```
Ассистент = LLM + Системный промпт + MCP-сервер (fsm-mcp)
```

Проводник построен на основе **FSM-архитектуры** — каждое состояние диалога описано в markdown-файле, а MCP-сервер выдаёт инструкции по запросу LLM.
