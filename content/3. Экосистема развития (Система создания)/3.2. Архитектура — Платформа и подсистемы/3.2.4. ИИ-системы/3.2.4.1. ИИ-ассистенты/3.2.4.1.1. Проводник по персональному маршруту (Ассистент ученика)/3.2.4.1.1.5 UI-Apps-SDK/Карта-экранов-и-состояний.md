## Состояния экранов

### 1. Главный экран (Home)

| Элемент | Описание |
|---------|----------|
| Заголовок | "Проводник — Ассистент ученика" |
| Описание | Краткое описание возможностей |
| CTA | Кнопка "Показать мою неделю" |
| Демо-переключатель | Выбор демо-профиля (Release 1) |

### 2. Экран загрузки (Loading)

| Элемент | Описание |
|---------|----------|
| Спиннер | Анимация загрузки |
| Текст | "Загружаем данные за неделю..." |
| Таймаут | 5 секунд, затем ошибка |

### 3. Экран результата (Success)

| Элемент | Описание |
|---------|----------|
| Период | "16 — 22 декабря 2025" |
| Метрики | 4 ключевые метрики |
| Рекомендация | 1-2 предложения |
| CTA | "Показать ещё раз" / "Выбрать другой профиль" |

### 4. Экран ошибки (Error)

| Элемент | Описание |
|---------|----------|
| Иконка | Предупреждение |
| Сообщение | "Не удалось загрузить данные" |
| Действие | "Попробовать снова" |

### 5. Экран пустых данных (Empty)

| Элемент | Описание |
|---------|----------|
| Иконка | Пустой календарь |
| Сообщение | "На этой неделе пока нет активности" |
| CTA | "Начать сейчас" (ссылка на курс) |

## Переходы между состояниями

```
Home ──[клик]──▶ Loading
Loading ──[успех]──▶ Success
Loading ──[ошибка]──▶ Error
Loading ──[пусто]──▶ Empty
Success ──[клик]──▶ Loading
Error ──[retry]──▶ Loading
Empty ──[клик]──▶ Home
```
