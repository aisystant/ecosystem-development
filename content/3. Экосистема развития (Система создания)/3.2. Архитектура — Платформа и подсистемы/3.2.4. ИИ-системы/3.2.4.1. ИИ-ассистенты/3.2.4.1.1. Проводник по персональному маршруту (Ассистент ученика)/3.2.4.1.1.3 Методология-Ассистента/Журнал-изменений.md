# Журнал изменений методологии

История изменений в методологии Проводника.

---

## [0.1.0] — 2025-12-23

### Добавлено
- Базовая версия методологии
- Принципы поведения ассистента
- Формат "один экран"
- Правила интерпретации метрик
- Запрещённые действия

### Метрики
- days_with_activity (0-7)
- total_minutes
- average_minutes_per_day
- slot_consistency (%)
- work_products_count

### Ограничения
- Только текущая неделя
- Только чтение данных
- Без персонализации (демо-режим)

---

## Планируемые изменения

### [0.2.0] — Планируется
- Добавить интерпретацию трендов (сравнение с прошлой неделей)
- Добавить типы рабочих продуктов в рекомендации
- Улучшить формулировки для edge cases

### [0.3.0] — Планируется
- Персонализация рекомендаций на основе истории
- Учёт целей пользователя
- Адаптивные слоты

---

## Формат записей

Каждое изменение описывается в формате:

```
## [версия] — дата

### Добавлено
- Новые функции

### Изменено
- Изменённое поведение

### Удалено
- Убранные функции

### Исправлено
- Баги и ошибки
```
