# Интерпретация недельных метрик

Правила, по которым Проводник интерпретирует данные и формулирует рекомендации.

## Метрика: Дни с активностью (days_with_activity)

| Значение | Интерпретация | Рекомендация |
|----------|---------------|--------------|
| 0 | Неделя без активности | "На этой неделе пока нет записей. Попробуйте начать с 10 минут чтения." |
| 1-2 | Эпизодическое участие | "Вы были активны X дня. Попробуйте добавить ещё один день на следующей неделе." |
| 3-4 | Регулярное участие | "Хороший ритм — X дня активности. Это уже формирующаяся привычка." |
| 5-7 | Стабильное участие | "X дней активности — это устойчивый ритм. Продолжайте в том же духе." |

## Метрика: Инвестированное время (total_minutes)

| Значение | Интерпретация | Рекомендация |
|----------|---------------|--------------|
| 0 | Нет данных | "Время пока не зафиксировано. Попробуйте таймер при занятии." |
| 1-30 | Минимальное участие | "X минут за неделю. Даже 15 минут в день дадут 100+ минут." |
| 31-120 | Регулярное участие | "X минут — это примерно Y минут в день. Хороший темп." |
| 121-300 | Активное участие | "X минут — солидное вложение. Вы явно нашли время для развития." |
| 300+ | Интенсивное участие | "Более 5 часов за неделю. Впечатляющий темп!" |

## Метрика: Систематичность слота (slot_consistency)

| Значение | Интерпретация | Рекомендация |
|----------|---------------|--------------|
| 0% | Нет привычки | "Регулярного времени не определено. Попробуйте выбрать слот." |
| 1-20% | Начало формирования | "Слот 'X' использован Y% дней. Привычка начинает формироваться." |
| 21-50% | Формируется привычка | "Слот 'X' — Y% дней. Ещё немного, и станет автоматическим." |
| 51-80% | Устойчивая привычка | "Слот 'X' — Y% дней. Это уже рабочая привычка." |
| 81-100% | Сильная привычка | "Слот 'X' — Y% дней. Отличная систематичность!" |

## Метрика: Рабочие продукты (work_products_count)

| Значение | Интерпретация | Рекомендация |
|----------|---------------|--------------|
| 0 | Нет продуктов | "Пока нет завершённых артефактов. Попробуйте написать заметку." |
| 1-2 | Начало | "X рабочих продукта. Каждый артефакт — это закреплённое понимание." |
| 3-5 | Продуктивная неделя | "X продуктов — хорошая продуктивность." |
| 6+ | Высокая продуктивность | "X продуктов — впечатляющий результат!" |

## Комбинированные паттерны

### Паттерн: "Много времени, мало продуктов"
```
total_minutes > 120 AND work_products_count < 2
```
**Рекомендация:** "Много времени, но мало артефактов. Попробуйте фиксировать мысли в заметках."

### Паттерн: "Много дней, мало времени"
```
days_with_activity >= 5 AND total_minutes < 60
```
**Рекомендация:** "Вы заходите часто, но по чуть-чуть. Попробуйте удлинить одну из сессий."

### Паттерн: "Сильный старт, потом спад"
```
actions_first_half > actions_second_half * 2
```
**Рекомендация:** "Активность в начале недели выше. К концу недели запал угасает?"
