# Словарь терминов проекта Проводник

## Основные термины

| Термин | Определение |
|--------|-------------|
| **Проводник** | ИИ-ассистент, сопровождающий ученика по персональной траектории развития |
| **Цифровой двойник** | Хранилище данных о действиях и прогрессе пользователя |
| **MCP** | Model Context Protocol — протокол связи между LLM и внешними инструментами |
| **Apps SDK** | OpenAI Apps SDK — платформа для создания приложений внутри ChatGPT |

## Метрики недели

| Термин | Определение |
|--------|-------------|
| **Дни с активностью** | Количество дней (0-7), когда пользователь совершал действия |
| **Инвестированное время** | Суммарное время в минутах за неделю |
| **Систематичность слота** | Доля дней (0-100%), когда пользователь занимался в запланированное время |
| **Рабочие продукты** | Артефакты, созданные пользователем (заметки, посты, ответы на практикумы) |

## Типы действий

| Термин | Определение |
|--------|-------------|
| **reading** | Чтение учебных материалов |
| **practice** | Выполнение практикумов |
| **writing** | Создание текстовых артефактов |
| **discussion** | Участие в обсуждениях |

## Технические термины

| Термин | Определение |
|--------|-------------|
| **twin_id** | Уникальный идентификатор Цифрового двойника пользователя |
| **demo_twin_id** | Фиксированный ID для демо-режима без идентификации |
| **MCP-инструмент** | Функция, доступная LLM для получения данных (например, `get_week_state`) |
| **Контракт** | JSON Schema, описывающая формат данных MCP-инструмента |

## Релизы

| Термин | Определение |
|--------|-------------|
| **Release 1** | Демо без идентификации — фиксированные данные |
| **Release 2** | Идентификация — реальные данные пользователя |
| **Release 3** | Безопасность — rate limiting, валидация |
