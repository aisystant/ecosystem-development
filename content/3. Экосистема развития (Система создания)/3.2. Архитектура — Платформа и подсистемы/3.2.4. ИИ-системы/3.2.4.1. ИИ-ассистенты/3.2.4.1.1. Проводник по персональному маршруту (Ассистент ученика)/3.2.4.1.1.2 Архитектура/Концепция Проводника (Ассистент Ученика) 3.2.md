# Концепция Проводника (Ассистент Ученика) 3.2

## Высокоуровневая концепция

Проводник — это **мост** между данными о прогрессе ученика и удобным интерфейсом для их осмысления.

```
┌─────────────────────────────────────────────────────────────┐
│                      ПРОВОДНИК                               │
│                                                              │
│   Данные             →    Интерпретация    →    Действие    │
│   (Цифровой двойник)      (ИИ-ассистент)       (Рекомендация)│
└─────────────────────────────────────────────────────────────┘
```

## Ключевые принципы

### 1. Данные — не оценка
Проводник показывает факты ("5 дней активности"), а не оценки ("отлично!").

### 2. Один экран
Вся информация помещается на один экран. Никакой прокрутки для понимания сути.

### 3. Конкретные рекомендации
Не "занимайтесь больше", а "попробуйте завтра 15 минут чтения".

### 4. Минимальный ввод
Пользователь нажимает одну кнопку — получает полную картину.

## Архитектурные решения

| Решение | Обоснование |
|---------|-------------|
| OpenAI Apps SDK | Интеграция в привычный интерфейс ChatGPT |
| MCP вместо прямого API | Стандартизация, безопасность, расширяемость |
| Cloudflare Workers | Быстрый ответ, глобальный edge |
| Неделя как период | Достаточно для паттернов, не перегружает |

### Release 2: Персонализация

- Защита от злоупотреблений
- Мониторинг и аналитика

## Границы системы

**Что входит в Проводник:**
- Отображение метрик недели
- Интерпретация данных
- Рекомендации на основе данных

**Что НЕ входит:**
- Сбор данных (это Цифровой двойник)
- Обучение (это курсы)
- Планирование (это будущие проекты)
