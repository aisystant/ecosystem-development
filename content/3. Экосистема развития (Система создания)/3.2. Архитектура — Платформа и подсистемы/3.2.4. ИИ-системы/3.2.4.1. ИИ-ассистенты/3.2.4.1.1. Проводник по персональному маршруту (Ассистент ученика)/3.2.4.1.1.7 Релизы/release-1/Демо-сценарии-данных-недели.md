## Как переключать сценарии

В демо-режиме (Release 1) переключение между сценариями можно реализовать:

1. **Вариант А:** Параметр в URL UI (`?demo=empty`, `?demo=weak`, `?demo=good`)
2. **Вариант Б:** Кнопки в UI для переключения демо-профиля
3. **Вариант В:** MCP принимает параметр `demo_profile` в запросе

Для приёмки использовать все три сценария последовательно.
