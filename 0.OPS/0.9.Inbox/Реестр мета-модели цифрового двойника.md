# Реестр мета-модели цифрового двойника

> **Версия:** 0.2

Этот документ содержит машиночитаемый реестр показателей цифрового двойника (`PRS.Twin`) с типами, форматами и порогами по ступеням.

---

## 1. Метаданные реестра

```json
{
  "version": "0.2",
  "updated": "2026-02-01",
  "notation": {
    "primary": "IND.1.*",
    "derived": "IND.2.*",
    "preferences": "IND.1.PREF.*"
  }
}
```

---

## 2. Ступени Ученика

| Код | Название | Период учёта |
|-----|----------|--------------|
| `STG.Student.Random` | Случайный | 1 неделя |
| `STG.Student.Practicing` | Практикующий | 4 недели |
| `STG.Student.Systematic` | Систематический | 8 недель |
| `STG.Student.Disciplined` | Дисциплинированный | 12 недель |
| `STG.Student.Proactive` | Проактивный | 24 недели |

---

## 3. Группы показателей

| Код группы | Название | Описание |
|------------|----------|----------|
| 1.PREF | Настройки и предпочтения | Настраиваемые параметры человека |
| 2.1 | Агентность и учебный ритм | Регулярность, часы, стабильность слотов |
| 2.2 | Степень мастерства | Интенсивность практики, качество продуктов |
| 2.3 | Характеристики по ступеням | Рефлексия, вопросы, управление вниманием |
| 2.4 | Квалификация | Ступени, история переходов |
| 2.5 | Мастерство по ролям | Ролевой профиль, индексы по ролям |
| 2.6 | Ресурсность и устойчивость | Ресурсность, нагрузка, ритм |
| 2.7 | Сообщество и вклад | Активность в клубе, помощь другим |
| 2.10 | Интегральные профили | Профиль застревания, риски |

---

## 4. Показатели: Настройки (IND.1.PREF.*)

Настраиваемые предпочтения человека.

### IND.1.PREF.daily_task_time

| Свойство | Значение |
|----------|----------|
| **Название** | Время получения задания |
| **Тип** | structural |
| **Формат** | time (HH:MM) |
| **Для промптов** | Да |
| **Пример** | `"08:00"` |

### IND.1.PREF.daily_report_time

| Свойство | Значение |
|----------|----------|
| **Название** | Время отчёта за день |
| **Тип** | structural |
| **Формат** | time (HH:MM) |
| **Для промптов** | Да |
| **Пример** | `"21:00"` |

### IND.1.PREF.role_set

| Свойство | Значение |
|----------|----------|
| **Название** | Текущие роли |
| **Тип** | semantic |
| **Формат** | list[string] |
| **Для промптов** | Да |
| **Пример** | `["менеджер", "отец", "студент"]` |

### IND.1.PREF.domain_focus

| Свойство | Значение |
|----------|----------|
| **Название** | Интересующие области |
| **Тип** | semantic |
| **Формат** | list[string] |
| **Для промптов** | Да |
| **Пример** | `["IT", "управление", "саморазвитие"]` |

### IND.1.PREF.objective

| Свойство | Значение |
|----------|----------|
| **Название** | Цели обучения |
| **Тип** | semantic |
| **Формат** | structured_text |
| **Для промптов** | Да |
| **Пример** | `"Освоить системное мышление для применения в работе"` |

## 5. Показатели: Агентность (IND.2.1.*)

### IND.2.1.1

| Свойство | Значение |
|----------|----------|
| **Название** | Среднее число часов саморазвития в неделю |
| **Тип** | temporal |
| **Формат** | float |
| **Единица** | hours/week |
| **Для квалификации** | Да |
| **Зависит от** | IND.1.3.4, IND.1.5.1 |

**Пороги по ступеням:**

| Ступень | Порог |
|---------|-------|
| Случайный | < 0.5 |
| Практикующий | ≥ 4 |
| Систематический | ≥ 6 |
| Дисциплинированный | ≥ 8 |
| Проактивный | ≥ 10 |

---

### IND.2.1.2

| Свойство | Значение |
|----------|----------|
| **Название** | Доля дней со слотом (регулярность) |
| **Тип** | frequency |
| **Формат** | float (0-1) |
| **Единица** | ratio |
| **Для квалификации** | Да |

**Пороги по ступеням:**

| Ступень | Порог (дней/неделю) | Ratio |
|---------|---------------------|-------|
| Случайный | 1 | < 0.14 |
| Практикующий | 3 | ≥ 0.43 |
| Систематический | 5 | ≥ 0.71 |
| Дисциплинированный | 6 | ≥ 0.86 |
| Проактивный | 6.7 | ≥ 0.95 |

---

### IND.2.1.4

| Свойство | Значение |
|----------|----------|
| **Название** | Устойчивость слотов (серии дней подряд) |
| **Тип** | frequency |
| **Формат** | integer |
| **Единица** | days |
| **Для квалификации** | Да |

**Пороги по ступеням:**

| Ступень | Порог (дней подряд) |
|---------|---------------------|
| Случайный | 0 |
| Практикующий | 3 |
| Систематический | 4 |
| Дисциплинированный | 5 |
| Проактивный | 6 |

---

### IND.2.1.6

| Свойство | Значение |
|----------|----------|
| **Название** | Индекс регулярности (интегральный) |
| **Тип** | scale |
| **Формат** | float (0-5) |
| **Для квалификации** | Да |
| **Вычисляется из** | IND.2.1.1, IND.2.1.2, IND.2.1.4 |

## 12. Расширение реестра

При добавлении нового показателя:

1. Добавить раздел с описанием в соответствующую группу (раздел 4-10)
2. Добавить в JSON-представление (раздел 11)
3. Обновить версию реестра
