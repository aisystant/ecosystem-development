## Инструкция для Claude

Чтобы прочитать нужную часть:
```
Read FPF-Spec.md с offset=<начальная_строка> limit=<количество_строк>
```

Например, для Part A:
```
offset=625, limit=11000
```

Для Part B:
```
offset=11836, limit=4000
```
