# Синергии характеристик (MHT-эффекты)

## 0. Назначение документа

Этот документ описывает **эмерджентные свойства** (синергии), возникающие при комбинации характеристик созидателя. Следует FPF B.2 (Meta-Holon Transition).

---

## 1. Концепция (FPF B.2)

### 1.1. Meta-Holon Transition (MHT)

**MHT** — переход, при котором комбинация частей создаёт новое целое с свойствами, которых нет у частей по отдельности.

```
CHR₁ + CHR₂ + ... + CHRₙ → Эмерджентное свойство
(при определённых пороговых значениях)
```

### 1.2. Условия синергии

- Каждая характеристика выше порогового значения
- Характеристики взаимодействуют (есть механизм связи)
- Результат > суммы частей

---

## 2. Каталог синергий

### 2.1. Лидерский потенциал

| Параметр | Значение |
|----------|----------|
| **Формула** | Агентность(≥4) + Калибр(≥4) + Коммуникативность(≥3) |
| **Эмерджентное свойство** | Способность вести людей к большим целям |
| **Маркеры** | Люди следуют, проекты масштабируются |
| **Связь с ролями** | Роль Просветитель |

### 2.2. Расширенный интеллект

| Параметр | Значение |
|----------|----------|
| **Формула** | Интеллект(≥4) + Техноинтеграция(≥4) |
| **Эмерджентное свойство** | Мышление, усиленное ИИ и экзокортексом |
| **Маркеры** | Решает задачи, недоступные без технологий |
| **Связь с ролями** | Роль Исследователь |

### 2.3. Резилентность

| Параметр | Значение |
|----------|----------|
| **Формула** | Стрессоустойчивость(≥4) + Агентность(≥3) + Самодисциплина(≥3) |
| **Эмерджентное свойство** | Антихрупкость — усиление от стресса |
| **Маркеры** | Быстрое восстановление, рост после неудач |
| **Связь с интегральными** | Антихрупкость |

### 2.4. Исследовательская инициатива

| Параметр | Значение |

| **Маркеры** | Публикации, открытия, новые методы |

## 3. Диагностика синергий

### 3.1. Чек-лист

Для каждой синергии:

1. ☐ Все компоненты выше порога?
2. ☐ Есть поведенческие маркеры?
3. ☐ Результаты превышают ожидания от суммы?

### 3.2. Пример

```
Проверка "Расширенный интеллект":
- Интеллект = 4 ✓
- Техноинтеграция = 4 ✓
- Маркер: использует Claude для сложных задач ✓
→ Синергия активна
```

---

## 4. Проектирование синергий

### 4.1. Стратегия развития

Вместо равномерного развития всех характеристик — **фокус на синергиях**:

1. Определить целевую синергию
2. Выявить недостающие компоненты (gap)
3. Целенаправленно развивать до порога
4. Проверить активацию синергии

### 4.2. Пример плана

```
Цель: активировать "Исследовательскую инициативу"

Текущее состояние:
- Интеллект: 3 (gap: +1)
- Агентность: 4 ✓
- Креативность: 2 (gap: +1)

План:
1. Курс МИМ → Интеллект +1
2. Практика brainstorming + ИИ → Креативность +1
3. Проверка: начать исследовательский проект
```
