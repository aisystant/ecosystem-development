# О связях характеристик

## 0. Назначение документа

Этот документ описывает **связи между характеристиками созидателя** — корреляции, синергии и эмерджентные эффекты. Следует FPF B.2 (Meta-Holon Transition).

---

## 1. Типы связей

### 1.1. Корреляции

**Корреляция** — статистическая связь между двумя характеристиками.

| Тип | Описание | Пример |
|-----|----------|--------|
| **Положительная** | Рост одной → рост другой | Агентность ↔ Самодисциплина |
| **Отрицательная** | Рост одной → снижение другой | (редко для наших характеристик) |
| **Нейтральная** | Нет связи | Физическая сила ↔ Креативность |

### 1.2. Причинные связи

**Причинная связь** — одна характеристика влияет на развитие другой.

```
A → B: развитие A способствует развитию B
```

### 1.3. Синергии (MHT)

**Синергия** — комбинация характеристик создаёт эмерджентное свойство, которого нет у отдельных характеристик.

## 3. Ключевые синергии (MHT-эффекты)

### 3.1. Лидерский потенциал

```
Агентность (≥4) + Калибр (≥4) + Коммуникативность (≥3)
→ Эмерджентное свойство: Лидерский потенциал
```

**Признаки:** способность вести людей к большим целям.

### 3.2. Расширенный интеллект

```
Интеллектуальная развитость (≥4) + Техноинтеграция (≥4)
→ Эмерджентное свойство: Расширенный интеллект
```

**Признаки:** мышление усилено ИИ и экзокортексом, "киборг-мышление".

### 3.3. Резилентность

```
Стрессоустойчивость (≥4) + Агентность (≥3) + Самодисциплина (≥3)
→ Эмерджентное свойство: Резилентность
```

**Признаки:** быстрое восстановление после неудач, антихрупкость.

### 3.4. Исследовательская инициатива

```
Интеллектуальная развитость (≥4) + Агентность (≥4) + Креативность (≥3)
→ Эмерджентное свойство: Исследовательская инициатива
```

**Признаки:** способность генерировать и проверять гипотезы самостоятельно.

## 4. Причинные цепочки

### 4.1. Цепочка развития созидателя

```
Агентность → Самодисциплина → Устойчивые слоты
→ Интеллектуальная развитость → Калибр личности
```

### 4.2. Цепочка техноусиления

```
Ресурсность (инструменты) → Техноинтеграция
→ Интеллектуальная развитость (экзокортекс)
→ Калибр личности (масштаб)
```
