# Протокол измерения характеристик

## 0. Назначение документа

Этот документ описывает **стандартный протокол измерения** каждой киберхарактеристики созидателя.

---

## 1. Общий протокол

### 1.1. Структура измерения

Для каждой характеристики:

```
1. Self-assessment (вес 0.3)
2. Поведенческие метрики (вес 0.4)
3. Экспертная оценка (вес 0.3)
→ Взвешенная оценка + confidence
```

### 1.2. Шкала оценки

| Уровень | Описание | Интерпретация |
|---------|----------|---------------|
| 1 | Критически низкий | Требуется срочная работа |
| 2 | Низкий | Есть проблема, нужно развитие |
| 3 | Достаточный | Базовый уровень достигнут |
| 4 | Хороший | Выше среднего, уверенное владение |
| 5 | Отличный | Мастерство, может обучать других |

---

## 2. Протоколы по характеристикам

### 2.1. Агентность

**Self-assessment (5 вопросов):**
1. Планирую ли день заранее? (0-1)
2. Есть ли устойчивые слоты саморазвития? (0-1)
3. Инициировал ли проект за последний месяц? (0-1)
4. Провожу ли регулярные рефлексии? (0-1)
5. Влияю ли на окружение (не только реагирую)? (0-1)

**Метрики:**
- % выполненных запланированных слотов (за 4 недели)
- Количество инициированных активностей

**Формула:**
```
Агентность = round(
  0.3 * (sum(self_q1..q5)) +
  0.4 * (slots_completion * 5) +
  0.3 * mentor_rating
)
```

### 2.2. Интеллектуальная развитость

**Self-assessment (5 вопросов):**
1. Освоил ли новый метод за последний месяц? (0-1)
2. Веду ли структурированный экзокортекс? (0-1)
3. Могу ли объяснить сложную концепцию новичку? (0-1)
4. Применяю ли системное мышление? (0-1)
5. Пишу ли рефлексивные тексты? (0-1)

**Метрики:**
- Количество записей в экзокортексе
- Количество написанных текстов/постов

**Артефакты:**
- Качество письменных работ (оценка 1-5)

### 2.3. Стрессоустойчивость

**Self-assessment (5 вопросов):**
1. Сохраняю ли продуктивность под давлением дедлайна? (0-1)
2. Как быстро восстанавливаюсь после стресса? (<24ч=1, >неделя=0)
3. Есть ли практики управления состоянием? (0-1)
4. Качество сна последний месяц? (хороший=1, плохой=0)
5. Сохраняю ли слоты развития под нагрузкой? (0-1)

**Метрики:**
- % сохранения слотов в периоды нагрузки
- Частота "срывов" (пропуск >3 дней подряд)

### 2.4. Ресурсность

**Self-assessment (5 вопросов):**
1. Сколько часов в неделю на саморазвитие? (≥10=1, <5=0)
2. Есть ли бюджет на образование? (да=1, нет=0)
3. Есть ли менторы/эксперты для консультаций? (≥2=1, 0=0)
4. Состою ли в профессиональных сообществах? (да=1, нет=0)
5. Есть ли необходимые инструменты? (да=1, нет=0)

**Метрики:**
- Фактические часы на развитие (трекинг)
- Размер сети контактов

### 2.5. Техноинтеграция

**Self-assessment (5 вопросов):**
1. Использую ли ИИ ежедневно? (да=1, нет=0)
2. Есть ли структурированная система заметок? (да=1, нет=0)
3. Автоматизирована ли рутина? (>50%=1, <20%=0)
4. Использую ли инструменты продуктивности? (да=1, нет=0)
5. Интегрированы ли инструменты между собой? (да=1, нет=0)

**Метрики:**
- Количество записей в экзокортексе
- Частота использования ИИ (сессии/день)

### 2.6. Калибр личности

**Self-assessment (5 вопросов):**
1. На какой горизонт планирую? (>1год=1, <месяц=0)
2. Сколько людей затрагивает моя работа? (>50=1, <5=0)
3. Есть ли публичная активность? (да=1, нет=0)
4. Руковожу ли командой/проектом? (да=1, нет=0)
5. Есть ли миссия/большая цель? (да=1, нет=0)

**Метрики:**
- Размер аудитории (подписчики, читатели)
- Масштаб текущего проекта

