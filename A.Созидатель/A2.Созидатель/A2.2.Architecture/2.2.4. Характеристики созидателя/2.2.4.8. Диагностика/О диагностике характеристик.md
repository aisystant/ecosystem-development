# О диагностике характеристик

## 0. Назначение документа

Этот документ описывает **методы диагностики характеристик созидателя** — как измерять, оценивать и отслеживать характеристики.

---

## 1. Цели диагностики

| Цель | Описание | Частота |
|------|----------|---------|
| **Первичная оценка** | Определение текущего состояния | При входе |
| **Мониторинг** | Отслеживание прогресса | Еженедельно/ежемесячно |
| **Gap-анализ** | Сравнение с целевым профилем | При планировании |
| **Валидация** | Подтверждение перехода на уровень | При переходе роли |

---

## 2. Методы диагностики

### 2.1. Self-assessment (самооценка)

| Параметр | Значение |
|----------|----------|
| **Метод** | Опросник по каждой характеристике |
| **Достоинства** | Быстро, доступно, субъективный взгляд |
| **Недостатки** | Bias, переоценка/недооценка |
| **F-G-R** | F=2, G=self, R=0.4 |

### 2.2. Наблюдение наставника

| Параметр | Значение |
|----------|----------|
| **Метод** | Оценка на основе поведенческих маркеров |
| **Достоинства** | Внешний взгляд, объективнее |
| **Недостатки** | Зависит от квалификации наставника |
| **F-G-R** | F=3, G=observed, R=0.6 |

### 2.3. 360° оценка

| Параметр | Значение |
|----------|----------|
| **Метод** | Опрос окружения (коллеги, команда) |
| **Достоинства** | Множество точек зрения |
| **Недостатки** | Трудоёмко, требует доверия |
| **F-G-R** | F=3, G=social, R=0.7 |

### 2.4. Анализ артефактов

| Параметр | Значение |
|----------|----------|
| **Метод** | Оценка результатов работы (тексты, проекты) |
| **Достоинства** | Объективные данные |
| **Недостатки** | Не все характеристики отражаются |
| **F-G-R** | F=4, G=artifacts, R=0.7 |

### 2.5. Метрики поведения

| Параметр | Значение |
|----------|----------|
| **Метод** | Автоматический сбор данных (слоты, активность) |
| **Достоинства** | Объективно, непрерывно |
| **Недостатки** | Не все характеристики измеримы |
| **F-G-R** | F=5, G=behavioral, R=0.8 |

---

## 3. Протокол диагностики

### 3.1. Первичная диагностика

```
1. Self-assessment по всем 6 киберхарактеристикам (15 мин)
2. Наблюдение наставника на первой встрече
3. Анализ входных артефактов (если есть)
4. Синтез: взвешенная оценка
```

### 3.2. Регулярный мониторинг

```
Еженедельно:
- Метрики поведения (автоматически)
- Краткий self-check (5 вопросов)

Ежемесячно:
- Полный self-assessment
- Сессия с наставником

Ежеквартально:
- 360° оценка (опционально)
- Gap-анализ относительно целевой роли
```

## 4. Инструменты

### 4.1. Экспресс-тест киберхарактеристик

**30 вопросов** (5 на каждую характеристику), шкала 1-5.

| Характеристика | Примеры вопросов |
|----------------|------------------|
| Агентность | "Планирую ли я день заранее?", "Инициирую ли проекты?" |
| Интеллект | "Освоил ли новый метод за месяц?", "Веду ли экзокортекс?" |
| Стресс | "Сохраняю ли слоты под нагрузкой?", "Как быстро восстанавливаюсь?" |
| Ресурсность | "Сколько часов в неделю на развитие?", "Есть ли менторы?" |
| Техноинтеграция | "Использую ли ИИ ежедневно?", "Есть ли система заметок?" |
| Калибр | "На какой горизонт планирую?", "Скольких людей затрагивает моя работа?" |

### 4.2. Карточка состояния

