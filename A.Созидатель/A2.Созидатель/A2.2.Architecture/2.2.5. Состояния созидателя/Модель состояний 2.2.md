# Модель состояний созидателя

## 0. Назначение документа

Этот документ описывает модель состояний созидателя — систему для описания текущего состояния человека.

---

## 1. Концепция модели

**Связь с характеристиками:**

Состояние системы — это совокупность текущих значений показателей её характеристик.

```
Характеристики → Показатели → Значения → Состояние
```

---

## 2. Категории состояний

### 2.1. Продуктивное состояние (2.2.5.1)

**Целевое операционное состояние** для работы.

Шкала: 0-5 баллов

**Подробнее:** [[Продуктивное состояние]]

### 2.2. Эмоциональные состояния (2.2.5.2)

Эмоции как состояния: радость, тревога, вдохновение, раздражение и др.

**Подробнее:** [[Эмоциональные состояния]]

### 2.3. Ролевые состояния (2.2.5.3)

Состояния исполнения ролей — какую роль человек исполняет сейчас.

**Подробнее:** [[Ролевые состояния]]

### 2.4. Состояния неудовлетворённости (2.2.5.4)

Мотивационные дефициты — ощущение нехватки чего-либо, запускающее изменения.

**Подробнее:** [[Неудовлетворённости]]

### 2.5. Состояния намерения (2.2.5.5)

Готовность к действию — состояние «собираюсь сделать».

**Подробнее:** [[Намерения]]

### 2.6. Физические состояния (2.2.5.6)

Самочувствие — бодрость, усталость, голод, боль и др.

**Подробнее:** [[Физические состояния]]

### 2.7. Социальные состояния (2.2.5.7)

Статус и включённость в сообщество.

**Подробнее:** [[Социальные состояния]]

### 2.8. Когнитивные состояния (2.2.5.8)

Состояния мышления — фокус, рассеянность, перегрузка и др.

**Подробнее:** [[Когнитивные состояния]]

---

## 3. Использование модели

### 3.1. Для самодиагностики

Оценка текущего состояния для принятия решений о деятельности.

### 3.2. Для наставничества

Понимание состояния ученика для адаптации подхода.

### 3.3. Для цифрового двойника

Отслеживание динамики состояний.
