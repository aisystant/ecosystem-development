# Ценность для созидателя

## 0. Назначение документа

Этот документ описывает **какую ценность получает созидатель** от участия в экосистеме и как эта ценность конвертируется в результаты его жизни. Документ является концептуальной «шапкой» раздела 2.1.

**Семейство F4:** Целевая × Предприниматель — «Ценность и бизнес-модели»
**Главный вопрос:** Какую ценность получает созидатель?

---

## 1. Системная модель ценности

### 1.1. Три измерения ценности

Созидатель получает ценность в трёх измерениях:

```
┌─────────────────────────────────────────────────────────────────┐
│                    ТРИ ИЗМЕРЕНИЯ ЦЕННОСТИ                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  1. ЛИЧНОСТНОЕ ИЗМЕРЕНИЕ                                        │
│     ├── Системное мышление — способность видеть целое          │
│     ├── Мировоззрение — понимание своего места в мире          │
│     └── Агентность — способность влиять на свою жизнь          │
│                                                                 │
│  2. ПРОФЕССИОНАЛЬНОЕ ИЗМЕРЕНИЕ                                  │
│     ├── Мастерство — умение создавать ценные результаты        │
│     ├── Репутация — признанная экспертиза                      │
│     └── Доход — монетизация компетенций                        │
│                                                                 │
│  3. СОЦИАЛЬНОЕ ИЗМЕРЕНИЕ                                        │
│     ├── Сообщество — окружение единомышленников                │
│     ├── Влияние — способность менять системы вокруг            │
│     └── Наследие — вклад в развитие области                    │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 1.2. Цикл создания ценности

## 2. Ценность по ролям созидателя

### 2.1. Ценность для Ученика

| Что получает | Результат | Измеримый эффект |
|--------------|-----------|------------------|
| Персональный маршрут | Понятный путь развития | Нет хаоса в обучении |
| Базовые практики | Устойчивый ритм развития | 10+ часов в неделю на развитие |
| Мировоззрение | Понимание «зачем» | Осмысленные решения |
| Первые артефакты | Доказательства навыков | 3-5 артефактов за квартал |

### 2.2. Ценность для Интеллектуала

| Что получает | Результат | Измеримый эффект |
|--------------|-----------|------------------|
| Мыслительное мастерство | Способность решать сложные задачи | Рост качества работы |
| Применение в проектах | Перенос знаний в практику | +20-50% к доходу |
| Peer-сообщество | Обратная связь и поддержка | Ускорение развития в 2-3 раза |
| Портфолио | Доказательства компетенций | 5-10 проектов |

### 2.3. Ценность для Профессионала

| Что получает | Результат | Измеримый эффект |
|--------------|-----------|------------------|
| Прикладное мастерство | Предсказуемые результаты | Premium-ставки |
| Независимость | Диверсифицированный доход | 30-50% дохода вне найма |
| Право наставничества | Доход от обучения других | Дополнительный поток |
| Участие в проектах | Revenue share | Доля в продуктах |

### 2.4. Ценность для Исследователя

| Что получает | Результат | Измеримый эффект |
|--------------|-----------|------------------|
| Исследовательское мастерство | Создание нового знания | Публикации, методы |
| Гранты и поддержка | Ресурсы для исследований | $500-2000/мес |
| Экспертный статус | Признание в области | Приглашения, цитирования |
| Лицензирование | Пассивный доход от разработок | Роялти |

### 2.5. Ценность для Просветителя

| Что получает | Результат | Измеримый эффект |
|--------------|-----------|------------------|
| Просветительское мастерство | Способность масштабировать влияние | Публичность |
| Платформенный доход | Доля от экосистемы | Значительный |
| Институциональное влияние | Создание школ, направлений | Наследие |
| Стратегическая позиция | Советы директоров, попечительство | Влияние на отрасль |

## 6. Принципы раздела 2.1

### 6.1. Ценность через результат, а не контент

Созидатель платит не за «доступ к курсам», а за **результат трансформации**:
- Артефакты вместо сертификатов
- Компетенции вместо часов просмотра
- Репутация вместо дипломов

### 6.2. Ценность масштабируется с ролью

Чем выше роль созидателя, тем больше ценность:
- Ученик получает ритм и маршрут
- Интеллектуал — рост дохода
- Профессионал — независимость
- Исследователь — влияние на область
- Просветитель — наследие

### 6.3. Ценность накапливается

Активы созидателя накапливаются и работают на него:

| Актив | Как накапливается | Как монетизируется |
|-------|-------------------|-------------------|
| Компетенции | Руководства + стажировки | Консультации, найм |
| Портфолио | Артефакты в проектах | Доказательства для клиентов |
| Репутация | Цифровой двойник + отзывы | Premium pricing |
| Сеть | Сообщество + коллаборации | Рекомендации, партнёрства |
| Контент | Публикации + курсы | Пассивный доход |

### 7.1. С разделом 1.2 (Контексты применения)

Раздел 1.2 описывает **где созидатель применяет мастерство**, раздел 2.1 — **какую ценность он из этого извлекает**.

Раздел 2.2 описывает **как устроен созидатель** (компетенции, роли), раздел 2.1 — **какую ценность он получает** от развития этих компетенций.

Раздел 2.3 описывает **как происходит развитие**, раздел 2.1 — **что созидатель получает** на каждом этапе этого пути.
