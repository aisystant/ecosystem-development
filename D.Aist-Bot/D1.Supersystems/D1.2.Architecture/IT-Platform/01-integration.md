# Интеграция бота в ИТ-платформу

## Точки интеграции

```
┌─────────────────────────────────────────────────────────┐
│                    C.IT-Platform                         │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐  ┌─────────────┐ │
│  │   LMS   │  │ Контент │  │   API   │  │ Инфраструкт.│ │
│  └────┬────┘  └────┬────┘  └────┬────┘  └──────┬──────┘ │
│       │            │            │              │         │
└───────┼────────────┼────────────┼──────────────┼─────────┘
        │            │            │              │
        ▼            ▼            ▼              ▼
┌─────────────────────────────────────────────────────────┐
│                     D.Aist-Bot                           │
│  • Прогресс   • knowledge_   • Claude    • PostgreSQL   │
│    синхрон.     structure.     API         Railway      │
│                   yaml                                   │
└─────────────────────────────────────────────────────────┘
```

## Компоненты интеграции

| Компонент платформы | Интеграция с ботом |
|---------------------|-------------------|
| **LMS** | Синхронизация прогресса пользователя |
| **Контент-система** | Единая база knowledge_structure.yaml |
| **AI-сервисы** | Общий Anthropic Claude API |
| **Инфраструктура** | Railway/Render, PostgreSQL |

## Потоки данных

### От платформы к боту
- Профиль пользователя
- Контент уроков
- Настройки персонализации

### От бота к платформе
- Прогресс (завершённые уроки)
- Метрики активности
- Рабочие продукты пользователя
