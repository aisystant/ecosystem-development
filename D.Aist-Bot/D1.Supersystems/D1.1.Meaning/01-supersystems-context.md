# Контекст надсистем Aist-Bot

## Две надсистемы

Aist-Bot функционирует в контексте **двух надсистем**:

```
┌─────────────────────────────────────────┐
│  D1.1: IT-Platform (ИТ-платформа)       │
│  ∋ Aist-Bot как компонент инфраструктуры│
└─────────────────────────────────────────┘
                    ∩
┌─────────────────────────────────────────┐
│  D1.2: Cyberidentity (Киберличность)    │
│  ∋ Aist-Bot как часть цифрового я       │
└─────────────────────────────────────────┘
```

## Надсистема 1: IT-Platform (ИТ-платформа)

**Роль бота**: Компонент ИТ-инфраструктуры экосистемы Aisystant.

**Ценность для платформы**:
- Точка входа в экосистему (низкий порог через Telegram)
- Конверсия пользователей в Aisystant
- Синхронизация прогресса с LMS
- Общие AI-сервисы (Claude API)

**Связь с C.IT-Platform** (ядро C):
- Бот интегрируется с LMS
- Использует общую базу контента
- Хранит данные в общей инфраструктуре

## Надсистема 2: Cyberidentity (Киберличность)

**Роль бота**: Часть цифровой идентичности пользователя, элемент экзокортекса.

**Ценность для киберличности**:
- Структурирование саморазвития
- Накопление рабочих продуктов (заметки, рефлексии)
- Формирование системы мышления
- Интеграция с персональными инструментами (Obsidian, экзокортекс)

**Связь с A.Systems-Builder** (ядро A):
- Бот развивает созидателя
- Формирует мета-навыки (рефлексия, системность)
- Создаёт привычки практики

## Пересечение надсистем

| Аспект | IT-Platform | Cyberidentity |
|--------|-------------|---------------|
| **Фокус** | Инфраструктура | Пользователь |
| **Данные** | Прогресс, метрики | Рабочие продукты, заметки |
| **Интеграция** | LMS, API | Obsidian, экзокортекс |
| **Метрика успеха** | DAU, retention | Трансформация мышления |

## Влияние на архитектуру бота

Две надсистемы определяют **двойную ответственность** бота:
1. **Как компонент платформы** — надёжность, масштабируемость, интеграция
2. **Как часть киберличности** — приватность, персонализация, экспорт данных
