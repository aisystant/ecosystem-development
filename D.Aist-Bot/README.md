# Ядро D: Aist-Bot (Telegram-бот системного помощника)

## Обзор ядра

**SoI (Система интереса)**: Telegram-бот — персональный помощник для развития системного мышления через регулярную практику и сборку экзокортекса.

**Репозиторий бота**: [github.com/aisystant/aist_bot](https://github.com/aisystant/aist_bot)

## Две надсистемы бота

Бот функционирует в контексте **двух надсистем**:

### 1. ИТ-платформа Экосистемы (C2)

- Бот как **компонент платформы**
- Техническая интеграция, API, инфраструктура
- Синхронизация прогресса с профилем Aisystant
- Приоритет для режимов **Марафон** и **Лента**

### 2. Киберличность Созидателя

- Бот как **часть персонального экзокортекса**
- Интеграция с Obsidian, локальными заметками
- Построение инфраструктуры мышления
- Приоритет для режима **Экзокортекс**

## Текущее состояние

Бот функционирует в двух режимах:
1. **Марафон** — 14-дневная интенсивная программа
2. **Лента** — гибкий формат по выбранным темам

Планируется третий режим:
3. **Экзокортекс** — режим сборки системы мышления (Obsidian-first)

## Технологический стек

- Python 3.11 + aiogram 3.x
- Anthropic Claude API
- PostgreSQL
- Railway / Render
