# Концепция Aist-Bot

## Миссия

**Трансформировать случайных учеников в системных созидателей** через персонализированный маршрут обучения в Telegram.

## Ценностное предложение

Aist-Bot — AI-ассистент, который:

1. **Адаптирует обучение** под индивидуальный темп и интересы
2. **Формирует привычку** системного мышления через регулярные микро-практики
3. **Создаёт рабочие продукты** — не просто читать, а строить инфраструктуру мышления
4. **Отвечает на вопросы** в контексте системного подхода и FPF

## Проблемы, которые решает бот

| Проблема | Решение бота |
|----------|--------------|
| Учусь хаотично, нет системы | Структурированные режимы с прогрессом |
| Не понимаю, с чего начать | Персональный маршрут от простого к сложному |
| Забываю изученное | Spaced repetition, регулярные напоминания |
| Теория без практики | Каждый шаг = рабочий продукт |
| Сложно найти ответы | AI-ответы в контексте системного мышления |

## Три режима работы

### 1. Марафон (Marathon)

**Для кого:** Новички, желающие быстрого погружения

**Формат:** 14-дневная интенсивная программа
- Последовательное изучение системного подхода
- Ежедневные уроки + практические задания
- Рабочие продукты на выходе каждого дня

**Метафора:** Спринт — быстрый старт с дедлайнами

### 2. Лента (Feed)

**Для кого:** Занятые люди, предпочитающие гибкий график

**Формат:** Самостоятельный темп
- Выбор до 3 тем в неделю
- Три уровня глубины для каждой темы
- Адаптивное усложнение после 7 успешных тем

**Метафора:** Поток — непрерывное развитие без давления

### 3. Экзокортекс (Exocortex) — планируется

**Для кого:** Продвинутые пользователи, строящие систему мышления

**Формат:** Сборка инфраструктуры мышления
- Прогресс по артефактам, а не по прочитанным урокам
- Каждое задание = рабочий продукт (заметка, структура, процедура)
- Автопроверки, обучающие FPF
- Duolingo-подобный прогресс (Level → Step → Check)

**Метафора:** Конструктор — build, а не read

## Целевая аудитория

### Первичная аудитория
- Ученики Aisystant на начальных этапах
- Люди, интересующиеся системным мышлением
- Самоучки, ищущие структуру

### Вторичная аудитория
- Наставники (мониторинг прогресса учеников)
- Методологи (тестирование контента)

## Ключевые метрики успеха

| Метрика | Описание |
|---------|----------|
| Retention D7/D30 | Возврат пользователей через 7/30 дней |
| Completion Rate | % завершивших режим до конца |
| Artifacts Created | Количество созданных рабочих продуктов |
| Question Quality | Релевантность ответов на вопросы |
| NPS | Net Promoter Score |

## Интеграция в экосистему

```
Созидатель (A2)
      │
      │ использует для обучения
      ▼
Aist-Bot (D2)
      │
      │ входит в
      ▼
ИТ-платформа (C2)
      │
      │ часть
      ▼
Экосистема Aisystant (B2)
```

Бот — точка входа в экосистему. Через бота созидатель:
1. Знакомится с системным мышлением
2. Формирует базовые компетенции
3. Переходит к более глубоким программам Aisystant
