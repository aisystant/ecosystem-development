# Режимы работы Aist-Bot

## Два основных режима

### 1. Марафон (Marathon)

**Определение**: 14-дневная структурированная программа обучения системному мышлению.

**Характеристики**:
- Фиксированная структура: 14 дней × 2 урока = 28 тем
- Строгий порядок прохождения
- Ежедневные уроки (теория) и задания (практика)
- Рабочие продукты как результат каждого задания

**Участник**: Ученик (Student)

**Траектория**:
```
День 1-7: Диагностика → Первые практики
День 8-14: Практика → Система → Роль созидателя
```

### 2. Лента (Feed)

**Определение**: Гибкий формат обучения по выбранным темам.

**Характеристики**:
- До 3 тем одновременно
- Дайджесты в удобном ритме
- Персональные заметки
- Нет строгого порядка

**Участник**: Читатель (Reader)

## Сравнение режимов

| Аспект | Марафон | Лента |
|--------|---------|-------|
| **Структура** | Фиксированная | Гибкая |
| **Длительность** | 14 дней | Без ограничений |
| **Темы** | 28 тем последовательно | До 3 тем параллельно |
| **Единица** | Урок + Задание | Дайджест |
| **Результат** | Рабочие продукты | Персональные заметки |
| **Участник** | Ученик | Читатель |

## Переключение между режимами

Команда `/mode` позволяет переключаться между режимами с сохранением прогресса.

## Планируемый режим: Экзокортекс

**Статус**: В разработке

**Концепция**: Режим сборки персональной системы мышления с интеграцией в Obsidian.
