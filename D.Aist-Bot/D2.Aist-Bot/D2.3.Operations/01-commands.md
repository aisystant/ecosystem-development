# Команды Aist-Bot

## Основные команды

| Команда | Описание | Режим |
|---------|----------|-------|
| `/start` | Начало работы, настройка профиля | Все |
| `/learn` | Получить следующий урок или дайджест | Все |
| `/progress` | Недельная статистика | Все |
| `/mode` | Переключение между режимами | Все |
| `/profile` | Просмотр настроек | Все |
| `/help` | Справка по командам | Все |

## Команды настройки

| Команда | Описание |
|---------|----------|
| `/language` | Выбор языка (RU/EN) |
| `/timezone` | Настройка часового пояса |
| `/difficulty` | Уровень сложности |
| `/notifications` | Настройка уведомлений |

## Flow настройки при `/start`

```
1. Выбор языка (RU/EN)
2. Настройка часового пояса
3. Выбор уровня сложности
4. Выбор режима (Марафон/Лента)
5. [Лента] Выбор до 3 тем
```

## Интерактивные элементы

Бот использует inline-кнопки для:
- Навигации между уроками
- Ответов на вопросы
- Подтверждения действий
- Выбора тем (Лента)

## Обработка ответов

Пользователь может отвечать:
- Текстом (свободный ответ)
- Голосовым сообщением (транскрибируется)
- Нажатием кнопок (быстрый выбор)
