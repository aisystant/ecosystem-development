## Слайд 8: Контур развития

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│      КОНТУР РАЗВИТИЯ — ЭТО ЗАМКНУТЫЙ ЦИКЛ               │
│                                                         │
│              ┌──────────┐                               │
│              │   ЦЕЛЬ   │                               │
│              └────┬─────┘                               │
│        ┌──────────┘                                     │
│        ▼                                                │
│   ┌──────────┐              ┌──────────────┐            │
│   │ ПРАКТИКА │───────────►  │    ЗАМЕР     │            │
│   └──────────┘              └──────┬───────┘            │
│                                    │                    │
│                                    ▼                    │
│   ┌──────────────┐         ┌──────────────┐            │
│   │ ОБНОВЛЁННАЯ  │◄────────│  КОРРЕКЦИЯ   │            │

│   └──────────────┘                                      │
```

**Время:** 3 мин

```
┌─────────────────────────────────────────────────────────┐
│                                                         │

│                                                         │

│  ──────────                  ──                         │

│  Датчик измеряет ───►        Замер ───►                 │

│       ↻                           ↻                     │

│  = котёл вхолостую     = работаете вслепую              │

```

**Время:** 3 мин
**Скрипт:** «Термостат не нуждается в мотивации. Он просто замкнут. Ваш контур развития — тот же принцип.»

## Слайд 10: Почему затухает

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│     ПОЧЕМУ «НАЧИНАЮ И БРОСАЮ»?                           │
│                                                         │
│        ┌──────────┐                                     │
│        │   ЦЕЛЬ   │                                     │
│        └────┬─────┘                                     │
│             │  ✗ РАЗРЫВ                                 │
│             ▼                                           │
│        ┌──────────┐         ┌──────────┐               │
│        │ ПРАКТИКА │── ✗ ──► │  ЗАМЕР   │               │
│        └──────────┘         └────┬─────┘               │
│                                  │  ✗ РАЗРЫВ           │
│                                  ▼                      │
│        ┌──────────┐         ┌──────────┐               │
│        │   ЦЕЛЬ   │◄── ✗ ──│КОРРЕКЦИЯ │               │
│        └──────────┘         └──────────┘               │
│                                                         │
│     Один разрыв = контур разомкнут = затухание          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**Время:** 2 мин
**Скрипт:** «Вот почему три сценария из начала. Рывок мотивации = разрыв между целью и практикой. Вечная занятость = разрыв между практикой и замером. Тупик = разрыв между коррекцией и целью.»
