## Слайд 8: Контур развития

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│      КОНТУР РАЗВИТИЯ — ЭТО ЗАМКНУТЫЙ КОНТУР              │
│                                                         │
│              ┌──────────────┐                            │
│              │     ЦЕЛЬ     │                            │
│              └──────┬───────┘                            │
│                     │                                    │
│                     ▼                                    │
│   ┌─────────────────────┐    ┌───────────────────────┐  │
│   │     ДЕЙСТВИЕ        │───►│       ЗАМЕР            │  │
│   │   (по Методу)       │    │ (Рабочий продукт?)    │  │
│   └─────────────────────┘    └──────────┬────────────┘  │
│                                         │               │
│                                         ▼               │
│   ┌─────────────────────┐    ┌───────────────────────┐  │
│   │   ОБНОВЛЁННАЯ       │◄───│     КОРРЕКЦИЯ         │  │
│   │      ЦЕЛЬ           │    │  (Метода / Цели)      │  │
│   └─────────────────────┘    └───────────────────────┘  │

│   Замкнут = самоподдерживается                          │
│   Разорван = затухает                                   │

```

**Время:** 3 мин
**Скрипт:** «Обратите внимание: не просто "делать", а действовать ПО МЕТОДУ. Не просто "замерять", а спрашивать — какой РАБОЧИЙ ПРОДУКТ остался? И корректировать не просто "подход", а конкретный МЕТОД. Если контур замкнут — развитие самоподдерживается. Если разорван — затухает. И никакая дисциплина не спасёт.»

│  │           │ нагрева       │ конспект.→корр. │          │
│  ├────────────────────┼──────────────────────┤          │
│  │ РП        │ 19° из 22°    │ Посты по        │          │
│  │           │               │ материалу       │          │

**Время:** 3 мин
**Скрипт:** «Термостат не нуждается в мотивации. Он просто замкнут. Обратите внимание: котёл работает по АЛГОРИТМУ — это метод. Датчик измеряет конкретную температуру — это рабочий продукт. Регулятор меняет РЕЖИМ котла — это коррекция метода. Ваш контур развития — тот же принцип.»

## Слайд 10: Почему затухает

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│     ПОЧЕМУ «НАЧИНАЮ И БРОСАЮ»?                           │
│                                                         │
│        ┌──────────────┐                                 │
│        │     ЦЕЛЬ     │                                 │
│        └──────┬───────┘                                 │
│               │  ✗ РАЗРЫВ (нет метода)                  │
│               ▼                                         │
│   ┌─────────────────┐    ┌──────────────────┐           │
│   │    ДЕЙСТВИЕ     │─✗─►│      ЗАМЕР       │           │
│   │  (по Методу)    │    │ (Рабочий продукт)│           │
│   └─────────────────┘    └────────┬─────────┘           │
│                                   │  ✗ РАЗРЫВ           │
│                                   ▼                     │
│   ┌─────────────────┐    ┌──────────────────┐           │
│   │  ОБНОВЛЁННАЯ    │◄─✗─│   КОРРЕКЦИЯ      │           │
│   │     ЦЕЛЬ        │    │  (Метода/Цели)   │           │
│   └─────────────────┘    └──────────────────┘           │
│                                                         │
│     Один разрыв = контур разомкнут = затухание          │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

**Время:** 2 мин
**Скрипт:** «Вот почему три сценария из начала. Рывок мотивации = нет метода, нет слота. Вечная занятость = нет рабочего продукта. Тупик = не меняете метод, цель устарела.»
