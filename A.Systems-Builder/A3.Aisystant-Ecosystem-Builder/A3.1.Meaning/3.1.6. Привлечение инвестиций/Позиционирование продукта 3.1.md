# Позиционирование продукта: два варианта

## 0. Зачем два позиционирования

Один продукт, две истории. Выбор зависит от инвестора:

| Тип инвестора | Позиционирование | Что слышит |
|---------------|-----------------|------------|
| EdTech/AI VC | **A: AI-native learning platform** | Рынок, масштаб, unit economics |
| Impact/Education | **B: Методология + платформа + сообщество** | Миссия, трансформация, сообщество |

Оба описывают одну систему. Разница — в точке входа и акцентах.

---

## A. EdTech + AI (масштабируемый бизнес)

### Заголовок

**AI Exocortex: IDE для интеллекта с verified knowledge base и knowledge supply chain.**

### Рынок

- AI в образовании: $10B+ (2025), растёт 30%+ в год
- Lifelong learning + AI-enhanced: $3B+ сегмент
- Creator economy (эксперты с аудиторией): $100B+

### Позиционирование

```
         Персонализация
              ↑
              │
  Coursera    │         [AI Exocortex]
  Udemy       │
  ────────────┼────────────────────→ Verified
              │                      Knowledge
  ChatGPT     │         Duolingo
  (галлюцинирует)       (микро-сессии)
              │
```

- **vs Coursera/Udemy:** Адаптивная подача + Digital Twin, а не один контент для всех
- **vs ChatGPT:** Verified knowledge base (Pack), а не статистические галлюцинации
- **vs Duolingo:** Системное мышление и развитие, а не микро-навыки
- **vs Notion/Obsidian:** AI-подача + формализация, а не мёртвые заметки

### Ключевые метрики для инвестора

| Метрика | Почему важна |
|---------|-------------|
| Cost per user | Neon + CF Workers = ~$0.01/user/month |
| CAC | Telegram = органический рост + Stars = low friction |
| Retention | Digital Twin + персонализация → sticky |
| LTV | Подписка + growth path (T1→T5) |
| Expansion | Template-exocortex: каждый эксперт = новый канал |

### Flywheel

```
Эксперт создаёт Pack
    → Template-exocortex (бот + персонализация)
        → Ученики подписываются (Telegram Stars)
            → Данные в Digital Twin
                → Лучшая персонализация
                    → Больше retention
                        → Привлекает новых экспертов
```

## B. Методология + Сообщество (impact)

### Заголовок

**МИМ — методология + ИИ-платформа + сообщество для развития созидателей 17-25 лет.**

### Миссия

Каждый человек, который хочет развиваться — получает работающий инструмент, а не ещё один курс.

Развитие — это инженерная задача. Не «стараться сильнее». Найти, что сломано, и починить. AI-экзокортекс замыкает цикл: **цель → практика → замер → коррекция → уточнённая цель.** Если цикл замкнут — он работает сам, без волевых усилий.

### Три компонента

| Компонент | Что | Зачем |
|-----------|-----|-------|
| **Методология** | Системное мышление + программы развития | Содержание: чему учить и как |
| **ИИ-платформа** | Бот + экзокортекс + MCP + агенты | Инструмент: персональная подача + замер + коррекция |
| **Сообщество** | systemsworld.club + семинары + мастерские | Среда: практика с другими людьми |

### Impact-метрики

| Метрика | Что измеряет |
|---------|-------------|
| Transfer to practice rate | Что ученик делает иначе (не «прочитал») |
| Цикл замкнут? | У скольких учеников работает цикл цель→практика→замер→коррекция |
| Time to first insight | Как быстро ученик находит свой первый разрыв |
| Community contribution | Вклад учеников в сообщество |

### Theory of Change

```
Человек хочет развиваться
    → Попадает в экосистему (бот, семинар, сообщество)
        → Находит свой разрыв (диагностика через бота)
            → Получает точный инструмент (Pack → персональная подача)
                → Замыкает цикл (практика + замер + коррекция)
                    → Развивается без героизма
                        → Становится созидателем
                            → Вносит вклад в сообщество
```

## C. Когда какой использовать

| Ситуация | Позиционирование |
|----------|-----------------|
| Pitch event, demo day | **A** (EdTech + AI) — быстро, цифры, масштаб |
| Разговор с impact-фондом | **B** (Методология + Сообщество) — миссия, трансформация |
| Технический партнёр | **A** с акцентом на архитектуру (Pack, MCP, template) |
| Образовательная организация | **B** с акцентом на методологию и совместные программы |
| Angel investor | Зависит от бэкграунда: tech → **A**, education → **B** |
