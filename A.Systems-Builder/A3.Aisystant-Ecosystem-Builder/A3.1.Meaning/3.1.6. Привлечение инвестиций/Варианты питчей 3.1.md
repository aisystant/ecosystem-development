# Варианты питчей экосистемы

## 0. Ключевая аналогия: IDE для интеллекта

Программист без IDE пишет код в блокноте. Программист с IDE — в среде, которая подсвечивает ошибки, подсказывает, собирает, тестирует. Разница — не в человеке, а в инструменте.

AI-экзокортекс — это **IDE для интеллекта**: среда, которая помогает человеку думать, учиться и развиваться — не за него, а вместе с ним.

| IDE (код) | Экзокортекс (интеллект) |
|-----------|------------------------|
| Editor | Pack (редактирование знаний) |
| Compiler | Extractor (формализация → сущности) |
| Linter | FPF/SPF (рамки корректности) |
| Debugger | Assessment + Refinement (диагностика) |
| Build system | Projection pipeline (Pack → Downstream) |
| Package manager | MCP (распределение знаний) |
| Version control | Git + Synchronizer |
| Project templates | Template-exocortex (fork & deploy) |
| Runtime | Бот (исполнение для конечного пользователя) |

**IDE = Integrated Development Environment.**
**Продукт = Integrated Development Environment for Intellect.**

## 1. Elevator Pitch (30 секунд, универсальный)

> ИИ-системы отвечают за людей, но не учат думать. Курсы дают контент, но не адаптируются. Заметки копят пыль.
>
> Мы строим **IDE для интеллекта** — AI-экзокортекс, где формализованные знания (не галлюцинации) подаются персонально через цифрового двойника. Бот в Telegram — точка входа. Template-экзокортекс — масштабирование: любой эксперт запускает свою мастерскую за день.
>
> 5300+ документов проиндексировано, 9 источников знаний, работающий бот с марафоном и лентой.

## 2. Питчи по аудиториям

### 2.1 Массовый пользователь

> **ИИ-система, которая учит тебя думать — не за тебя, а вместе с тобой.**
>
> Бот в Telegram, который каждый день даёт тебе одну мысль из проверенной базы знаний, задаёт вопрос, помогает применить к твоей ситуации. Не ChatGPT, который отвечает на всё и ни за что не отвечает. А система, которая знает, что ты уже знаешь, и ведёт дальше.

### 2.2 Техническая аудитория

> **AI Exocortex: IDE для интеллекта с формализованной базой знаний и агентным циклом.**
>
> Pack-система (structured knowledge по SPF-стандарту) + MCP-серверы (knowledge + digital-twin + guides) + AI-агенты (Extractor, Strategist, Synchronizer) + Telegram-бот как runtime. Знания хранятся в Git, индексируются в Neon pgvector, подаются через CF Workers AI. Template-экзокортекс — fork & deploy за 1 час.

### 2.3 EdTech-инвесторы

> **AI-native learning platform с verified knowledge base, адаптивной персонализацией через цифрового двойника и knowledge supply chain.**
>
> Проблема: LLM-тьюторы галлюцинируют, курсы one-size-fits-all, PKM мертвы без привычки. Решение: формализованные знания (Pack, не промпт) + персональная подача (Digital Twin) + платформа для экспертов (Template-exocortex). Moat: Pack-система не копируется промптом — это structured domain knowledge с верификацией.
>
> Модель: подписка B2C (Telegram Stars) + платформа B2B (эксперты создают мастерские). Neon + CF Workers = копеечный hosting, низкий CAC через Telegram.

### 2.4 Impact-инвесторы

> **Методология + ИИ-платформа + сообщество для развития созидателей.**
>
> МИМ (Мастерская Интеллектуального Мастерства) — экосистема, где люди 17-25 лет учатся системному мышлению через практику. ИИ-экзокортекс замыкает цикл: цель → практика → замер → коррекция. Не лайфхаки, а инженерный подход к развитию. Бот — точка входа, сообщество — среда, методология — содержание.
>
> Миссия: каждый человек, который хочет развиваться — получает работающий инструмент, а не ещё один курс.

### 2.5 Партнёры (эксперты)

> **Создайте свою мастерскую на нашей инфраструктуре: Pack → Bot → Community за 1 день.**
>
> Вы — эксперт в своей области. У вас есть знания, курсы, книги. Мы даём инфраструктуру: формализуете знания в Pack (формат SPF), разворачиваете Template-экзокортекс, получаете бота с персонализацией и аналитикой. Ваши ученики получают AI-проводника, который знает вашу методологию — а не галлюцинирует.
